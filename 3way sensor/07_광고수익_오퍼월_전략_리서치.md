## 2. 광고 네트워크 비교: AdMob vs 카카오 vs 기타

### Google AdMob (1순위 — 필수 연동)

**AdSense는 웹용, AdMob이 앱용** (혼동 주의)

| 광고 형식 | 한국 eCPM | 1회당 수익 | 사용자 경험 영향 |
|----------|---------|----------|---------------|
| **리워드 동영상** | $17~29 (₩23,000~39,000/천회) | **~₩33/회** | 최소 (자발적 시청) |
| 인터스티셜 (전면) | $10~11 (₩13,500~15,000/천회) | ~₩14/회 | 중간 (화면 전환 시) |
| 네이티브 | $1.5~8 | ~₩4/회 | 최소 (콘텐츠 사이) |
| 배너 | $0.2~1.0 | ~₩0.5/회 | 항상 노출 |

### 기타 광고 네트워크

| 네트워크 | 유형 | 특징 | 권장도 |
|---------|------|------|--------|
| **카카오 애드핏** | 배너/네이티브 | 국내 광고주 풍부, 카카오 DAU 기반 | ★★★★ |
| **AppLovin MAX** | 미디에이션 | 여러 네트워크 자동 경쟁 입찰 → eCPM +20~40% | ★★★★ |
| **ironSource** | 미디에이션+오퍼월 | 게임 앱 최강, 팬플러스 성공 사례 | ★★★ |
| 애드팝콘 | 오퍼월/리워드 | 국내 3,500만+ 유저 데이터 | ★★★ |

### 미디에이션이란?

```
[광고 지면 1개]
    ↓
[미디에이션 플랫폼 (MAX or AdMob Mediation)]
    ├── AdMob: "이 지면에 ₩25 낼게"
    ├── 카카오 애드핏: "₩18 낼게"
    └── Meta: "₩30 낼게"  ← 자동으로 이 광고 선택
```

여러 광고 네트워크를 동시에 연결하여 **가장 높은 단가를 자동 선택**. 단일 네트워크 대비 eCPM 20~40% 향상.

## 3. 오퍼월(Offerwall) 상세

### 오퍼월이란?

앱 안에 있는 **"미션 게시판"**. 사용자가 광고주 미션(앱 설치, 카드 발급, 보험 견적 등)을 완료하면 포인트를 받고, 라브는 광고주로부터 CPA/CPE를 받는 구조.

```
[사용자] → 오퍼월에서 미션 선택 → 미션 완료
                                      ↓
[광고주] → CPA 단가 지급 → [오퍼월 플랫폼] → 수수료 공제 → [라브 수취]
                                                              ↓
                                                     [사용자에게 SSP 포인트 지급]
```

### CPA 단가 범위 (한국 기준)

| 미션 유형 | 광고주 단가 | 라브 수취 (60~70%) | 사용자 SSP |
|-----------|-----------|------------------|----------|
| 앱 설치 (CPI) | ₩500~3,000 | ₩300~2,100 | 50~200 SSP |
| 앱 설치 + 레벨업 (CPE) | ₩3,000~30,000 | ₩1,800~21,000 | 200~2,000 SSP |
| 금융상품 가입/카드 발급 | ₩20,000~100,000 | ₩12,000~70,000 | 1,000~5,000 SSP |
| 보험 견적 (리드) | ₩3,000~15,000 | ₩1,800~10,500 | 200~1,000 SSP |

**오퍼월 eCPM은 배너/동영상보다 월등히 높음**: 글로벌 $30~$530, 한국 팬플러스 앱 Android 기준 **$342**

### 한국 오퍼월 플랫폼 비교

| 순위 | 플랫폼 | 진입장벽 | Flutter 연동 | 특징 |
|------|--------|---------|------------|------|
| **1순위** | **애디슨 AdiSON (엔비티)** | 중간 (파트너센터 셀프 연동) | 네이티브 브릿지 필요 | 국내 No.1, 네이버웹툰/토스/OK캐쉬백 파트너 |
| **2순위** | **NAS 오퍼월 (MAFIN)** | **매우 낮음** (회원가입 즉시) | **WebView로 즉시 가능** | GitHub 오픈소스 샘플, 리워드 비율 직접 설정 |
| **3순위** | TNK Factory | 중간 | WebView 방식 | CPI/CPA/CPS, 전담 매니저 |
| **4순위** | 버즈빌 BuzzBenefit | 높음 (MAU 10만+ 권고) | 네이티브 브릿지 | 잠금화면+오퍼월 통합, B2B 솔루션 |

## 4. 라브 자체 오퍼월 구축 가능성

### 결론: 직접 구축은 비효율적

| 구성 요소 | 직접 구축 시 필요한 것 | 현실성 |
|---------|-------------------|----|
| 광고주 영업 | CPA 캠페인 직접 유치 | ❌ 영업 인력 필요 |
| 트래킹 SDK | 전환 추적 서버 | ⚠️ 개발 가능하나 복잡 |
| 사기 방지 | 봇/가짜전환 필터 | ❌ 전문 기술 필요 |
| 광고 인벤토리 | 캠페인 수 확보 | ❌ 최소 수백 개 필요 |
| **예상 비용** | **6개월+ 개발 + 영업팀** | **비현실적** |

### 권장: 기존 오퍼월 SDK 연동 (2~4주 구현)

```
라브 앱 → NAS 오퍼월 WebView 또는 애디슨 네이티브 SDK
       → 사용자가 미션 완료
       → 오퍼월 서버 → 라브 서버에 포스트백(callback) 전송
       → 라브 서버: SSP 포인트 적립
       → 사용자: SSP로 네이버페이/카카오 교환
```

## 5. 라브 앱 광고 배치 전략

### 라브에 적합한 광고 배치

| 광고 형식 | 배치 위치 | 수익 기여 | UX 영향 |
|----------|---------|---------|--------|
| **리워드 동영상** | "오늘의 보너스 SSP" — 홈에서 광고 보고 SSP 추가 적립 | **높음** | **없음** (자발적) |
| **리워드 동영상** | 활동 완료 후 "광고 보고 2배 SSP 받기" | **높음** | 없음 |
| **오퍼월** | "미션으로 SSP 모으기" 전용 탭 | **매우 높음** | 없음 (별도 탭) |
| 네이티브 광고 | 활동 기록 리스트 사이에 자연스럽게 | 중간 | 최소 |
| 배너 | 탄소 대시보드 하단 | 낮음 | 약간 |
| 인터스티셜 | ❌ 사용 안 함 | - | **높음** (UX 해침) |

### 핵심 원칙

```
✅ "광고를 보면 SSP가 더 쌓인다" — 사용자 선택권
❌ "강제로 광고를 봐야 앱을 쓸 수 있다" — 절대 금지

라브 = 활동 기반 SSP 적립이 핵심
광고 = SSP를 빠르게 모으는 "부스터" 역할
오퍼월 = SSP를 대량으로 모으는 "미션센터" 역할
```

| **합계** | | **약 7~9일** | Week 7-8 내 수용 가능 |

### 구현 아키텍처

```
[Flutter 앱]
├── 홈 화면
│   └── "오늘의 보너스" 버튼 → 리워드 동영상 (AdMob)
│
├── 활동 완료 화면
│   └── "광고 보고 2배 SSP" → 리워드 동영상 (AdMob)

└── 탄소 대시보드
    └── 하단 배너 (카카오 애드핏 또는 AdMob)

[API 서버]
├── POST /api/ad/reward-callback  ← AdMob 서버 검증
│   └── SSP 적립 (시청 확인 후)
│
├── POST /api/offerwall/callback  ← NAS/애디슨 포스트백
│   └── SSP 적립 (미션 완료 확인)
│
└── GET /api/ad/config
    └── 광고 표시 여부, SSP 배수 등 설정 (관리자 조정 가능)
```

## 9. 결론 및 권장 전략

### Phase 1 광고 수익 전략 (즉시 실행)

```
1순위: AdMob 리워드 동영상 (자발적 시청 → SSP 부스터)
       → 가장 쉽고, UX 해치지 않으며, 단가 높음 (₩33/회)
       → Flutter 공식 패키지로 1-2일 연동

2순위: NAS 오퍼월 (미션센터 탭)
       → WebView로 즉시 연동 가능, 진입장벽 최소
       → CPA 단가가 가장 높은 수익원 (₩1,000~70,000/건)

3순위: 카카오 애드핏 배너 (대시보드 하단)
       → 수익 기여는 적지만, 항상 노출되어 기본 수익 확보
```

### Phase 2 이후 확장

```
MAU 5만+ 달성 시:
├── 애디슨 AdiSON 프리미엄 오퍼월로 업그레이드
├── AppLovin MAX 미디에이션 적용 (eCPM +20~40%)
└── B2B 챌린지 광고 (기업이 자사 챌린지에 광고비 지불)

MAU 10만+ 달성 시:
├── 버즈빌 BuzzBenefit 잠금화면 광고 도입 검토
├── 프리미엄 퀴즈형 광고 (캐시워크 모델)
└── 자체 광고 영업 (스포츠 브랜드 직접 계약)
```
