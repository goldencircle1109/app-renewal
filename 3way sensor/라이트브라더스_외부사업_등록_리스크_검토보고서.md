## 1. 검토 요약

### 결론

**라이트브라더스의 현행 방법론(MC10-ver01)은 POPLE 자발적 레지스트리 전용으로 개발된 것이며, 배출권거래제 외부사업(KOC)으로 직접 등록하는 데는 구조적·제도적 리스크가 높다.**

주요 리스크 요인은 다음 5가지로 요약된다:

| 순위 | 리스크 | 심각도 | 극복 난이도 |
|:---:|--------|:------:|:----------:|
| 1 | POPLE 방법론 ≠ K-ETS 승인 방법론 (제도 불일치) | **치명적** | 높음 |
| 2 | 추가성(Additionality) 입증 한계 | **높음** | 매우 높음 |
| 3 | 설문 기반 베이스라인의 MRV 취약성 | **높음** | 중간 |
| 4 | 감축량 규모 대비 경제성 부족 | **중간** | 중간 |
| 5 | 정책 변동 및 이중계산 리스크 | **중간** | 낮음 |

## 2. 탄소배출권 시장 구조와 외부사업의 위치

### 2.1 글로벌 탄소배출권 시장의 3층 구조

탄소배출권 시장은 법적 구속력과 운영 주체에 따라 3개 층위로 구분된다.

```
┌─────────────────────────────────────────────────────────────────┐
│  1층: UNFCCC/파리협정 기반 국제 규제시장                            │
│  ├─ CDM CER (교토의정서 체제, 사실상 신규 등록 종료)                 │
│  ├─ Article 6.2 ITMO (당사국 간 양자 협정 기반)                    │
│  └─ Article 6.4 PA6.4ER (CDM 후속 메커니즘, 구축 중)               │
├─────────────────────────────────────────────────────────────────┤
│  2층: 국가 규제시장 (Compliance Market)                            │
│  ├─ 한국 K-ETS: KAU(할당배출권), KOC(외부사업), i-KOC(국외)         │
│  ├─ EU ETS: EUA (외부 상쇄 크레딧 사용 제한)                       │
│  └─ 기타 국가 ETS (중국, 일본 등)                                  │
├─────────────────────────────────────────────────────────────────┤
│  3층: 자발적 탄소시장 (Voluntary Carbon Market)                    │
│  ├─ Verra VCS → VCU (세계 최대, 시장 점유율 70~80%)               │
│  ├─ Gold Standard → GS-VER (SDG 연계 프리미엄)                    │
│  ├─ ACR → ERTs, CAR → CRTs                                      │
│  └─ 국내: POPLE, 한국형 자발적 탄소시장(K-VCM, 2026~ 구축 중)       │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 각 시장 간 관계와 상호 전환

| 전환 경로 | 가능 여부 | 조건 |
|----------|:--------:|------|
| CDM CER → i-KOC | **가능** | CDM Registry에서 CER 자발적 취소 + 증빙 제출 후 전환 |
| VCU/GS-VER → KOC | **불가** | 자발적 크레딧은 K-ETS 법적 경로 없음 |
| **POPLE → KOC** | **불가** | POPLE은 국내 자발적 레지스트리, K-ETS 인정 불가 |
| KOC → KCU | **가능** | 상쇄배출권으로 전환 후 의무이행 제출 (상한 10%) |
| 국내 외부사업 → CORSIA | **불가** | KOC는 CORSIA 인정 대상 아님 |

### 2.3 POPLE 레지스트리의 위치

**POPLE (Promise for Our Planet)**은 한국형 자발적 탄소 레지스트리로, 배출권거래제의 법적 체계와는 별개로 운영된다.

| 구분 | K-ETS 외부사업 (KOC) | POPLE |
|------|----------------------|-------|
| 법적 근거 | 배출권거래법 제29~30조 | 없음 (민간 자발적 체계) |
| 크레딧 효력 | 할당대상업체 의무이행 사용 가능 | 자발적 상쇄·마케팅용 |
| 방법론 승인 | 인증위원회 심의·고시 등재 필수 | POPLE 자체 심사 |
| 검증 기관 | 국가표준원 인정 검증기관 | POPLE 지정 기관 |
| 크레딧 가격 | KCU 연동 (~7,000~12,000원/톤) | 시장 미형성 (저가) |
| 수요처 | 배출권 할당대상업체 (법적 수요) | 기업 ESG·자발적 탄소중립 |

> **핵심 시사점:** 라이트브라더스의 현행 방법론은 POPLE 등록용이므로, K-ETS 외부사업으로 전환하려면 **방법론 재개발 + 인증위원회 승인**이라는 별도의 절차가 필요하다.

### 2.4 한국 배출권거래제 외부사업 등록 요건

외부사업으로 등록되기 위한 4대 핵심 요건 (배출권거래법 제30조, 환경부 고시 제2023-258호):

| 요건 | 설명 |
|------|------|
| **추가성** | 타 법령 의무 이행이 아닌, 자발적 추가 조치임을 입증 |
| **정량성·검증 가능성** | 지속적·정량적 측정 가능, 독립 제3자 검증 통과 |
| **승인 방법론 적용** | 인증위원회 승인 방법론 또는 CDM 방법론 사용 |
| **이중계산 금지** | 타 제도 중복 인증 불인정 |

### 2.5 파리협정 체제와 외부사업의 미래

- **Article 6.2 (ITMO):** 정부 간 양자 협정 기반, 상응조정(Corresponding Adjustment) 의무
- **Article 6.4:** CDM 후속 메커니즘, 2024년 이후 본격 구축 중
- **제4차 계획기간 (2026~2030):** 외부사업 승인 기준 강화 — 파리협정 국내 발효일(2016.12.3) 이후 시작 사업만 인정, 실태조사 대상 확대

### 2.6 자발적 탄소시장 동향

- **글로벌 VCM 규모:** 2024년 약 40억 달러, 2030년 240억~1,200억 달러 전망
- **ICVCM Core Carbon Principles (CCP):** 10대 핵심 원칙 기반 품질 인증 — CCP 라벨 크레딧은 평균 25% 가격 프리미엄
- **한국형 자발적 탄소시장 (K-VCM):** 기획재정부 주도 거래소 신설 추진 (2026년 기준·발행체계 구축, 2027년 시범사업 예정)
- **CORSIA 연계:** Verra VCS, Gold Standard 등 8개 프로그램만 인정, K-ETS KOC는 미인정

## 3. 방법론 분석

### 3.1 방법론 구조 요약

```
감축량(EB) = 베이스라인 배출량(BE) - 사업 배출량(PE)
```

#### 베이스라인 배출량 (BE)
- **산식:** `BE_y = D_A × TEF_A`
- **의미:** 자전거로 주행한 거리만큼 기존 교통수단(자가용)을 운행했을 때 발생했을 배출량
- **배출계수 (TEF_A):** 유종별 고정값 사용

| 유종 | 배출계수 (kg CO₂eq/km) |
|------|:---------------------:|
| 휘발유 | 0.15871 |
| 경유 | 0.19701 |
| LPG | 0.18212 |

#### 사업 배출량 (PE)
- **산식:** `PE_A = PE_maintenance,A + PE_vehicle_usage,b,y`
- **구성:** 자전거 제조·수리 배출 + 공공자전거 재배치 배출(전력 + 화석연료)

#### 감축량 산정
- **산식:** `EB_A = BE_A - PE_A`
- **누출(Leakage):** 고려하지 않음

### 3.2 방법론의 핵심 특성

| 항목 | 내용 | 리스크 평가 |
|------|------|:----------:|
| 대체 교통수단 범위 | **자가용만** (대중교통 제외) | 보수적 ✅ |
| 주행거리 측정 | 측정값 우선, 불가시 통계값 (불확도 적용) | 적절 ✅ |
| 기존 교통수단 확인 | **매 주행 시 설문조사** | 취약 ⚠️ |
| 배출계수 출처 | IPCC 기본 배출계수 + 국가 고유 순발열량 | 적절 ✅ |
| 공공자전거 이중등록 방지 | 타 감축사업 미등록 증명 필요 | 적절 ✅ |
| 누출 | 미고려 | 리스크 ⚠️ |

## 4. 외부사업 등록 리스크 분석

### 4.1 [치명적] 제도적 불일치: POPLE 방법론 ≠ K-ETS 승인 방법론

**문제의 본질:**

현행 방법론(MC10-ver01)은 POPLE 자발적 레지스트리 등록을 목적으로 개발되었으며, 배출권거래제 외부사업의 법적 요건을 충족하지 않는다.

| K-ETS 외부사업 요건 | 현행 방법론 충족 여부 |
|-------------------|:-------------------:|
| 인증위원회 승인 방법론 사용 | ❌ POPLE 자체 방법론 |
| CDM 방법론 준용 | ❌ 별도 체계 |
| 국가표준원 인정 검증기관 검증 | ❌ POPLE 자체 검증 |
| 상쇄등록부(ORS) 등록 | ❌ POPLE Registry |

**극복 경로:**
1. 국내 승인 방법론으로 재개발하여 인증위원회 심의 통과 (소요: 1~3년)
2. CDM AMS-III.BM 방법론을 준용하여 국내 적용 (전례: 한국교통안전공단의 공유자전거 방법론)

**참고:** 서울시 따릉이가 외부사업 승인(2024.03)을 받을 수 있었던 것은, 한국교통안전공단이 별도로 개발하여 상쇄등록부시스템에 등록한 「공유자전거 도입을 통한 교통수단 대체 사업의 방법론」을 적용했기 때문이다.

### 4.2 [높음] 추가성(Additionality) 입증 한계

추가성이란 "해당 사업이 없었더라면 감축이 발생하지 않았을 것"을 입증하는 것으로, 외부사업 등록의 가장 큰 허들이다.

#### 라이트브라더스 방법론의 추가성 리스크 요인

**(1) 개인의 자발적 교통수단 선택 문제**
- 자전거 주행은 개인의 건강·경제·편의 동기에 의한 자발적 행위
- "탄소 크레딧이 없었어도 자전거를 탔을 것"이라는 반론에 대한 방어가 매우 어려움
- 특히 자전거 통근·출퇴근은 경제적 인센티브(유류비 절감)가 이미 존재

**(2) 정부 정책과의 경계 모호성**
- 자전거 이용 활성화는 정부의 녹색교통 정책 기본 방향과 일치
- 「자전거 이용 활성화에 관한 법률」에 의한 자전거도로 확충, 공공자전거 보급 등이 이미 시행 중
- 정책적으로 장려되는 행위를 "추가적 감축"으로 인정받기 어려움

**(3) 기술 보급 확산에 따른 추가성 상실**
- 전기자전거·공유 모빌리티 서비스가 시장 표준으로 자리잡으면, 베이스라인 갱신 시 추가성 재입증이 불가능해질 수 있음

**(4) 공공자전거 vs 민간 플랫폼의 차이**
- 서울시 따릉이(공공자전거)도 추가성 논란을 거쳤으나, **공공 인프라 투자의 추가적 탄소감축 효과**로 논리를 구성
- 민간 자전거 플랫폼(라이트브라더스)은 영리 사업으로서 추가성 논증이 더욱 어려움

#### 추가성 입증 전략 (참고)

| 접근법 | 설명 | 적용 가능성 |
|--------|------|:----------:|
| 투자 분석 | 탄소 크레딧 수익 없이는 사업 수익성이 확보되지 않음을 입증 | 낮음 (자전거 앱은 독자 수익모델 보유) |
| 장벽 분석 | 제도적·기술적·문화적 장벽이 존재함을 입증 | 중간 |
| 일반 관행 분석 | 유사 사업이 시장에서 일반적이지 않음을 입증 | 중간 (모빌리티 시장 성장으로 약화 추세) |

### 4.3 [높음] 설문 기반 베이스라인의 MRV 취약성

#### 문제점

현행 방법론은 **"매 주행 시마다 설문조사 등을 통해 기존 교통수단을 확인"**하도록 규정하고 있으나, 이는 다음과 같은 구조적 취약성을 내포한다:

| 취약점 | 설명 |
|--------|------|
| **응답 편향 (Desirability Bias)** | 이용자가 감축 실적을 높이기 위해 "자가용 대체"로 과다 응답할 인센티브 존재 |
| **검증 불가능성** | 이용자가 실제로 해당 교통수단을 보유·이용했는지 독립 검증 불가 |
| **무응답·불성실 응답** | 매 주행 시 설문 참여율과 응답 품질 관리의 현실적 어려움 |
| **대체 효과의 불확실성** | 자전거 이용이 없었을 때 정말 자가용을 운행했을지 반사실적(counterfactual) 증명 불가 |

#### 외부사업 기준과의 괴리

K-ETS 외부사업 MRV 기준(환경부 고시 제2023-258호)은 **"지속적이고 정량화된 측정"**과 **"제3자 검증기관에 의한 독립 검증"**을 요구한다. 설문 기반 데이터는:

- **측정의 객관성:** GPS 추적 데이터 → 높음 / 설문 응답 → 낮음
- **제3자 검증 가능성:** 설문 원본 데이터 → 검증기관이 응답의 진위를 확인할 방법 없음
- **국제 기준 비교:** CDM AMS-III.BM도 설문조사를 허용하나, **통계적 유의성 확보를 위한 최소 샘플 크기, 무작위 추출, 연간 갱신** 등 엄격한 통계 요건을 부과

#### 개선 방향

| 현행 | 개선 안 |
|------|---------|
| 매 주행 시 설문조사 | GPS 기반 자동 주행거리 측정 + 교통카드/차량등록 데이터 교차 검증 |
| 이용자 자기보고 교통수단 | 차량 보유 여부 사전 등록 + 주기적 샘플 검증 |
| 개별 주행 단위 산정 | 통계적 샘플링 기반 연간 평균 산정 |

### 4.6 방법론 고유 취약점 상세

#### (1) 누출(Leakage) 미고려

현행 방법론은 "누출은 고려하지 않습니다"라고 명시하고 있으나, 외부사업 심의에서는 누출 항목이 중요 검토 대상이다.

**잠재적 누출 시나리오:**
- 자전거 제조 과정에서의 해외 탄소배출 (공급망 누출)
- 자전거 이용 증가에 따른 도로 인프라 변경의 간접 배출
- 리바운드 효과: 자전거 이용으로 절약한 연료비를 다른 탄소 배출 활동에 사용

CDM AMS-III.BM은 소규모 방법론 특례로 누출 미계산을 허용하나, K-ETS 외부사업에서는 누출 분석을 요구할 수 있다.

#### (2) 대중교통 대체 제외의 양면성

- **장점:** 보수적 접근으로 감축량 과대 산정 방지
- **단점:** 실제 자전거 이용자 중 상당수가 대중교통 이용자를 포함 → 자가용 대체 비율이 낮아지면 실효 감축량이 미미해질 수 있음

#### (3) 베이스라인 고정 기간 미명시

K-ETS 외부사업은 일반적으로 7년(갱신) 또는 10년 고정 중 선택해야 하나, 현행 방법론에는 베이스라인 고정 기간이 명시되어 있지 않다.

## 6. 자발적 탄소시장 vs 규제시장 경로 비교

라이트브라더스가 탄소크레딧을 활용할 수 있는 현실적 경로는 3가지다.

### 경로별 상세 비교

| 항목 | 경로 A: K-ETS 외부사업 (KOC) | 경로 B: Verra VCS (VCU) | 경로 C: Gold Standard (GS-VER) |
|------|---------------------------|----------------------|------------------------------|
| **시장 성격** | 국가 규제시장 | 글로벌 자발적 시장 | 글로벌 자발적 시장 (프리미엄) |
| **주요 수요자** | 국내 할당대상업체 | 글로벌 기업, CORSIA 항공사 | SDG 중시 글로벌 바이어 |
| **크레딧 단가** | ~7,000~12,000원/톤 | $3.5~$14.8/톤 (품질별) | $10~$30+/톤 (프리미엄) |
| **방법론** | 신규 개발 필요 (또는 따릉이 방법론 준용) | AMS-III.BM/BJ 적용 가능 | CDM 방법론 준용 가능 |
| **등록 비용** | 5,000만~1억 원 | $10,000~$50,000+ | $15,000~$70,000+ |
| **등록 기간** | 1~3년 (방법론 승인 포함) | 6개월~1년 | 6개월~1.5년 |
| **추가성 허들** | 매우 높음 | 높음 | 높음 |
| **CORSIA 연계** | 불가 | 가능 | 가능 |
| **K-ETS 활용** | 직접 가능 | 불가 | 불가 |
| **SDG 라벨** | 없음 | 제한적 | 강점 (프리미엄 형성) |

### 현실적 권장 순서

```
1단계 (단기): POPLE 유지 + 한국형 자발적 탄소시장(K-VCM) 참여 준비
  └─ 현행 방법론으로 POPLE 크레딧 발행 지속
  └─ K-VCM 거래소 출범(2026~2027) 시 선점 효과

2단계 (중기): K-ETS 외부사업 등록 추진
  └─ 따릉이 사례의 승인 방법론을 참조하여 방법론 재개발
  └─ 이용자 규모 확대 후 경제성 확보 시점에 추진

3단계 (장기): 글로벌 자발적 시장 진출
  └─ Verra VCS 또는 Gold Standard 등록
  └─ 해외 기업 Scope 3 상쇄 수요 타겟
```

## 7. 한국 자발적 탄소시장 미활성화 원인 분석

### 7.1 글로벌 시장 규모 비교

| 구분 | 규모 (2024년) | 시장 점유율 | 주요 특징 |
|------|:------------:|:---------:|----------|
| 글로벌 VCM 전체 | 약 40억 달러 | 100% | 2030년까지 연평균 35.1% 성장 전망 |
| 북미 (미국 중심) | 약 14.8억 달러 | 37.1% | 최대 시장, S&P 500 기업 60% 이상 참여 |
| 유럽 | 약 7.2억 달러 | 18% | 유럽 기업이 글로벌 구매의 47% 차지 |
| 한국 | 약 1.1억 달러 추정 | 약 2.7% | 17개 프로젝트, 120만 tCO₂eq (2024.10 기준) |

한국의 VCM 시장은 미국·유럽 대비 **10~60배 이상의 규모 격차**가 존재한다.

### 7.2 미국·유럽의 자발적 탄소시장 활성화 요인

#### 제도적 기반

- **미국:** 캘리포니아 Cap-and-Trade, RGGI 등 주(州) 단위 탄소시장이 가격 발견 메커니즘 학습 기반 제공. CFTC가 VCM 크레딧을 상품(commodity)으로 분류하여 기관 투자자 참여 확대 추진
- **유럽:** EU 택소노미, CSRD(기업 지속가능성 보고 지침)가 탄소 공시를 법적 의무화. "그린워싱 지침(Green Claims Directive)"을 통해 탄소중립 주장 시 검증된 크레딧 사용 의무화

#### ESG 및 기업 문화

- SBTi 등록 유럽 기업 1,400개 이상 → Scope 3 잔여 배출량 상쇄를 위해 VCM 크레딧 구매
- 미국 S&P 500 기업의 60% 이상이 탄소시장을 ESG 전략에 활용
- Microsoft, Google, Apple 등 빅테크의 대규모 자발적 구매가 시장을 견인
- BlackRock, Vanguard 등 기관 투자자의 기후 공시 요구 → VCM 참여 간접 강제

#### 시장 인프라

- **거래소:** CBL(Xpansiv), 나스닥 카본 마켓 등 전문 거래소의 실시간 가격 공시와 유동성 제공
- **평가사:** Sylvera, BeZero Carbon 등 탄소 크레딧 등급 평가사가 정보 비대칭 해소
- **생태계:** 전문 브로커, 핀테크 플랫폼, ESG 데이터 제공사 등 인프라 성숙

### 7.3 한국 VCM 미활성화의 구체적 원인

#### (1) 제도적 원인: K-ETS와의 구조적 충돌

K-ETS라는 강력한 규제시장이 오히려 VCM 발전을 억제하는 **역설적 구조**가 존재한다.

| 문제 | 설명 |
|------|------|
| **이중계산 규제 공백** | VCM 크레딧은 K-ETS에서 법적 활용 불가. NDC 연계 시 이중계산 방지 규정도 불명확 |
| **VCM 전용 법률 부재** | 탄소중립기본법은 K-ETS 중심 설계. VCM 참여 기업의 권리·의무 관계 법적 불명확 |
| **규제 완화 압력** | 경총이 K-ETS에 국제 VCM 크레딧 활용 허용 요청 → 정책 방향 혼선 |
| **자발적탄소시장연합회** | 2024년 3월에야 출범 — 미국(2000년대)·유럽(2010년대) 대비 15~20년 뒤처짐 |

> **핵심 역설:** 유럽은 EU-ETS와 VCM을 **병행 발전**시킨 반면, 한국은 K-ETS **편중**으로 VCM 생태계가 형성될 공간이 협소했다.

#### (2) 시장 구조적 원인: 수요-공급 미스매치

**수요 측면:**
- K-ETS 대상 기업들은 할당 배출권 내 운영으로 추가 자발적 상쇄 구매 유인 부재
- 대다수 기업의 탄소 비용 관리가 규제 준수(compliance)에 집중
- RE100 참여 기업도 VCM 크레딧보다 REC(재생에너지 인증서) 구매 선호

**공급 측면:**
- 국내 VCM 프로젝트 17개, 약 120만 tCO₂eq에 불과 (글로벌 대비 미미)
- 산림·해양 블루카본 등 잠재 자원은 있으나 방법론 미비

**가격 발견 메커니즘 부재:**

## 8. 방법론 고도화 전략: 국제 인증 및 외부사업 등록 로드맵

### 8.1 현행 방법론 한계 진단

| 항목 | 현행 MC10 | K-ETS/국제 인증 요구 수준 |
|------|-----------|--------------------------|

| 모니터링 방식 | 매 주행 후 설문 | GPS·앱 자동 로그 + 제3자 검증 가능 데이터 |

| 이중계산 방지 | 타 감축사업 미등록 증명만 요구 | 레지스트리 간 상호 검증 체계 |
| SDG 임팩트 | 없음 | 정량화 및 제3자 검증 필수 (Gold Standard 의무) |

### 8.2 K-ETS 외부사업(KOC) 승인을 위한 개선 방안

#### MRV 고도화: 설문에서 GPS 기반 자동 모니터링으로 전환

```
[사용자 앱] → GPS 주행 로그 자동 수집 (정확도 ±10m, 샘플링 10초)
    ↓
[데이터 전처리 서버]
  - 최소 주행거리 필터 (500m 이상)
  - 자전거 주행 속도 범위 검증 (4~30 km/h)
  - 중복 주행 제거, 출발지/목적지 클러스터링
    ↓
[교통 빅데이터 교차 검증]
  - T-money 교통카드 집계 데이터와 OD(기종점) 비교
  - T-map·카카오내비 집계 자동차 이동 패턴 비교
    ↓
[감축량 산정 엔진]
  - 연간 갱신 베이스라인 배출계수 적용
  - 보수적 할인율(10~20%) 자동 적용
    ↓
[검증기관 대시보드]
  - API를 통한 원시데이터 접근 + 이상 데이터 자동 플래그
    ↓
[레지스트리 발행] → POPLE / K-ETS ORS / Verra Registry
```

| 파라미터 | 현행 | 개선안 |
|---------|------|--------|
| 주행 거리 | 설문 응답 | GPS 자동 측정 (오차 ±5% 이내) |
| 대체 수단 | 설문 선택 | 출발지·목적지 패턴 + 교통카드 교차 검증 |
| 배출계수 | 고정값 | 연도별 에너지원단위 갱신 (국가 에너지통계) |
| 누출 | 미적용 | 앱 서버 전력소비, 자전거 제조 LCA 간이값 |

#### 추가성 입증 3단계 전략

| 분석 방법 | 핵심 논거 | 자료 소스 |
|----------|---------|----------|
| **투자분석** | 탄소크레딧 수익 없이는 앱 개발·운영비(서버, GPS 인프라, 검증비용) 회수 불가 → IRR/NPV 입증 | 재무제표, 사업계획서 |
| **장벽분석** | 자동차 중심 교통 인프라, 자전거 전용도로 부족, 우천·지형 등 이용 기피 요인 | 국가교통DB, 기상청 데이터 |
| **일반관행분석** | 출퇴근 목적 자전거 이용률 10% 미만 → "일반관행 아님" 입증 | 국가교통조사, 통계청 |

#### 베이스라인 설정 강화

**1단계 (단기): 층화된 베이스라인**
- 유종 구성비 가중평균 배출계수 적용 (현행 방식 유지 + 갱신 주기 설정)
- 이륜차(0.0799 kgCO₂eq/km), 택시(0.21 kgCO₂eq/km) 등 세분화

**2단계 (중기): 동적 베이스라인**
- 해당 지역의 연간 교통수단 분담률 조사 반영
- 교통 빅데이터(T-data, 내비게이션 데이터) 연계한 사업지역 특정 베이스라인 설정

#### 감축량 산정 공식 보완

현행: `EB = BE - PE`

개선안: `ER = BE - PE - LE` (K-ETS 외부사업 표준 구조)
- LE(누출) 항 추가: 앱 서버 전력소비, 자전거 제조 LCA 간이값

### 8.3 국제 인증(Verra VCS / Gold Standard) 등록 전략

#### CDM AMS-III.BM 정합성 확보

Verra는 CDM AMS-III.BM("Lightweight Two and Three Wheeled Personal Transportation")을 VCS 프로그램에 편입하여 운영하며, VMR0013 개정판이 발행되었다.

| AMS-III.BM 요건 | MC10 대응 방안 |
|----------------|---------------|
| 대상: 자전거, e-바이크, e-스쿠터 등 경량 2·3륜 | 앱 사용 자전거 명시적 포함 확인 |
| 사업유형: 자전거 차선, 공유 프로그램, 주차 인프라 도입 | **앱+인프라 패키지형 사업으로 재설계** |
| 모니터링: 이용자 수, 주행거리, 주행 행태 추적 | GPS 기반 자동 모니터링으로 업그레이드 |
| 규모 제한: 60 kt CO₂eq/연 이하 (소규모) | 초기 규모 설계 시 반영 |

> **핵심 이슈:** AMS-III.BM은 **인프라 도입** 프로젝트를 전제한다. 순수 앱 기반 행동변화 사업은 적용 범위 해석에 불확실성이 있으므로, 앱 + 자전거 거치대 설치 + 안전 교육 등을 묶은 **패키지 사업**으로 재구성을 권장한다. Verra에 사전 문의(methodologies@verra.org)로 적용 가능성을 확인하는 것이 필수이다.

#### ICVCM Core Carbon Principles 충족

| CCP 원칙 | 핵심 요구사항 | MC10 대응 전략 |
|---------|-------------|---------------|
| 추적성 | 고유번호 부여, 이중계산 방지 | 사용자별 크레딧 직렬번호 시스템 |
| 강건한 정량화 | 보수적 산정, 불확실성 관리 | GPS 오차범위 명시, 보수적 할인율 적용 |
| 추가성 | 사전적 입증 | 투자/장벽/일반관행 분석 문서화 |
| 강건한 MRV | 제3자 독립 검증 | 공인 VVB(검증기관) 계약 의무화 |
| 지속가능개발 | SDG 기여 | 건강·대기질 공편익 정량화 |

#### Gold Standard SDG 임팩트 정량화

Gold Standard는 2024년 8월부터 디지털 SDG Impact Tool 사용이 의무화되었다.

| SDG 목표 | 지표 | 정량화 방법 |
|---------|------|----------|
| SDG 3 (건강) | 신체활동 증가로 만성질환 예방 | WHO 가이드라인 준수율 변화 측정 |
| SDG 7 (에너지) | 화석연료 절감량 | 대체 km × 유종별 에너지 소비량 |
| SDG 11 (지속가능도시) | 자전거 친화 인프라 이용률 | 전후 비교 설문 + GPS 데이터 |
| SDG 13 (기후행동) | CO₂eq 감축량 | 방법론 핵심 지표 |

### 8.4 디지털 MRV 및 블록체인 활용

| 레이어 | 기능 | 기술 스택 |

| PDD(프로젝트 설계 문서) 작성 | AMS-III.BM 템플릿 기반, 영문 | M24~M30 |
| VVB 선정 및 Validation | Verra 승인 VVB (SGS, Bureau Veritas 등) | M30~M36 |
| Verra 심의 및 등록 | VCS 레지스트리 등록 | M36~M40 |
| Gold Standard SDG 등록 (선택) | GS4GG 등록 + SDG Impact Tool 제출 | M36~M44 |
| 첫 Verification 및 크레딧 발행 | VCU 또는 GS-VER 발행 | M42~M48 |

### 8.6 즉시 실행 가능한 우선순위 액션 5가지

1. **Verra 방법론 적용 가능성 사전 문의 (즉시):** methodologies@verra.org에 사업 개요 송부하여 AMS-III.BM 적용 여부 확인
2. **에너지공단 외부사업 사전상담 신청 (즉시):** offset.energy.or.kr 상담 채널을 통해 교통 부문 신규 방법론 개발 가능성 타진
3. **GPS 데이터 수집 인프라 구축 착수 (1~3개월):** 현행 앱에 GPS 로그 수집 기능 추가 — 모든 인증 경로의 공통 전제조건
4. **추가성 분석 보고서 발주 (3~6개월):** 국내 교통 연구기관 또는 전문 컨설팅사에 투자/장벽/일반관행 분석 의뢰
5. **VVB 사전 접촉 (6개월):** K-ETS + Verra 양쪽 경험 있는 검증기관(SGS Korea, Bureau Veritas Korea, DNV Korea)과 방법론 개발 단계부터 협력

## 9. 3Way Sensor 신규기능 기반 방법론 고도화 가능성 분석

### 9.1 3Way Sensor 프로젝트 현황

**프로젝트명:** 3Way Activity Sensor / Wright Brothers 앱 회원 활동 자동 인식 기능
**현재 상태:** 연구·설계 완료, 프로덕션 코드 미구현 (2026.03.19 작성)

#### 기술 스택

| 항목 | 내용 |
|------|------|
| 지원 플랫폼 | Android (Kotlin) / iOS (Swift) / React Native / Flutter |
| 핵심 센서 | 가속도계(90% 역할), GPS(Phase 2), 자이로스코프, 만보계, 기압계 |
| 저장소 | SQLite (로컬) → REST API (서버 동기화) |
| ML 확장 | TensorFlow Lite / Core ML (Phase 3) |

#### 3단계 구현 로드맵

| Phase | 내용 | 자전거 인식 정확도 | 배터리 소모 |
|-------|------|:-----------------:|:----------:|
| Phase 1 (MVP) | 플랫폼 API 기반 활동 자동 감지 | 75% | 3% 이하/일 |
| Phase 2 | GPS 보정 + 센서 융합 | **90%** | 5% 이하/일 |
| Phase 3 | 커스텀 ML 모델 (온디바이스) | **95~98%** | 경량 |

#### 수집 가능 데이터

```sql
-- activity_records 테이블 (설계 완료)
activity_type   -- 'WALKING', 'RUNNING', 'CYCLING'
started_at      -- 활동 시작 시간
ended_at        -- 활동 종료 시간
duration_sec    -- 지속 시간 (초)
confidence      -- 신뢰도 (0~1.0)
steps           -- 걸음수
distance_m      -- 이동 거리 (미터)
avg_speed_kmh   -- 평균 속도
calories        -- 칼로리
```

| 교통수단 | 감지 난이도 | 방법 | 기대 정확도 |
|---------|:----------:|------|:---------:|
| 자전거 | ⭐⭐ | 플랫폼 API + GPS | 80~95% |
| 승용차 | ⭐⭐⭐ | GPS + 소거법 | 80~88% |
| 버스 | ⭐⭐⭐⭐ | GPS + GIS 노선 매칭 | 75~85% |

### 9.2 3Way Sensor가 MRV 고도화에 기여하는 방식

현행 방법론(MC10-ver01)의 핵심 약점과 3Way Sensor의 해결 능력을 매핑하면:

| 현행 약점 | 3Way Sensor 해결 방안 | 효과 |
|----------|---------------------|------|
| **설문 기반 주행거리** | GPS + 가속도계 자동 측정 → `distance_m` 필드 | 설문 제거, 객관적 데이터 확보 |
| **대체 교통수단 자기보고** | 승용차/버스/지하철 자동 감지(Phase 2~3) | "자동차 대신 자전거" 자동 입증 |
| **신뢰도 검증 불가** | `confidence` 필드(0~1.0) → 0.8 미만 데이터 자동 제외 | 데이터 품질 관리 내장 |
| **데이터 위변조 리스크** | 센서 원시 데이터 해시 + 서버 검증 | 제3자 검증 가능성 확보 |
| **누출 미고려** | 앱 서버 전력소비 자동 기록 가능 | 누출 항목 정량화 |

#### 핵심 시나리오: "자동차 대체" 자동 입증

```
[기존 방식] 설문: "오늘 자전거 대신 뭘 탔을 건가요?" → 자기보고 편향

[3Way Sensor 방식]
1. 평일 출퇴근 패턴 학습 (ML)
   - 과거 6개월: 월~금 08:00~09:00 승용차 감지 → "자가용 통근자"로 분류
2. 자전거 주행 감지
   - 2026-03-25 08:15 자전거 활동 감지 (confidence: 0.93)
   - GPS 경로: 집 → 직장 (7.2km)
3. 자동 대체 판정
   - "승용차 통근자가 자전거로 출퇴근" → 자동 감축량 산정
   - BE = 7.2km × 0.15871 kgCO₂eq/km = 1.14 kgCO₂eq
```

### 9.3 교통 빅데이터 API 연동 가능성

#### 주요 API별 연동 현황

| API | 인증 방식 | 일일 한도 | 비용 | MRV 활용 목적 |
|-----|----------|:--------:|:----:|-------------|
| **따릉이 실시간 API** | API Key | 1,000건/호출 | 무료 | 공공자전거 이동거리 직접 산출 |
| **T-map 자전거 경로** | API Key | 1,000건/일 | 무료→유료 | 자전거 vs 자동차 경로 거리 비교 |
| **ITS 교통량 API** | API Key | 10만 건/일 | 무료 | 구간별 자동차 통행량 베이스라인 |
| **기상청 단기예보** | API Key | 10만 건/일 | 무료 | 날씨 보정계수 (우천일 제외) |
| **티머니 API** | 제휴 계약 | 협의 | 협의 | 지역별 교통수단 분담률 통계 (보조적 활용) |
| **카카오모빌리티** | REST API Key | 30만 건/일 | 무료→유료 | 자동차 경로 탐색 (자전거 미지원) |
| **차량등록 정보** | API Key + 동의 | 표준 | 무료 | 자동차 보유 여부 확인 |

#### 티머니 API 상세

- **포털:** apiportal.tmoney.co.kr
- **접근 방식:** B2B 제휴 협약 기반 (무료 공개 아님)
- **제공 데이터:** 지하철 승하차, 시내버스 이용이력, 공공자전거, 택시 결제
- **MRV 활용:** 자전거 이용자의 이전 대중교통 이용 패턴과 교차 검증 → 모달 쉬프트(Modal Shift) 실적 검증
- **제약:** 개인식별 연계는 개인정보보호법 제약 → 집계 수준 데이터만 활용 가능

#### T-map API 상세 (MRV 핵심)

- **포털:** tmapapi.tmapmobility.com
- **핵심 기능:** **자전거 경로 API 제공** (카카오는 미지원)
- **MRV 활용 시나리오:**
  - 3Way Sensor GPS 출발지/도착지 → T-map 자동차 경로 거리 조회
  - 동일 구간 자전거 실제 주행 거리와 비교
  - **대체 거리 = 자동차 경로 거리** → 배출계수 적용 → 감축량 산정
- **비용:** 무료 1,000건/일, 초과 시 종량제

#### 공공자전거 API (따릉이)

- **데이터:** 실시간 대여정보, 대여이력, 대여소 현황
- **포털:** data.seoul.go.kr (서울 열린데이터광장)
- **MRV 활용:** 대여소 간 거리 → 실제 자전거 이동거리(km) 직접 산출
- **타 지역:** 대전 타슈(data.go.kr), 울산 피프틴 등 — 서울·대전이 API 품질 최우수

#### 기상청 API

- **포털:** data.go.kr (기상청 단기예보 조회서비스)
- **비용:** 완전 무료, 즉시 승인
- **MRV 보정:** 강수일·폭염일(35°C+)·대기오염 경보일 제외 → "자전거 이용 가능 유효일" 산정 → 통계적 신뢰도 향상

4. **외부사업 방법론 재개발 검토:** 한국교통안전공단의 「공유자전거 도입을 통한 교통수단 대체 사업의 방법론」을 참조하여 K-ETS 승인 방법론으로 재설계
5. **이용자 규모 확대:** 최소 5만 명 이상의 활성 이용자 확보 → 경제성 확보 기반 마련
6. **추가성 논증 전략 수립:** 투자 분석(탄소 크레딧 수익 필요성) + 장벽 분석(자전거 인프라 부족 지역 타겟) 병행

#### 장기 (2년 이상)

7. **글로벌 자발적 시장 진출 검토:** Verra VCS 또는 Gold Standard 등록으로 글로벌 기업의 Scope 3 상쇄 수요 타겟
8. **ICVCM CCP 라벨 획득 준비:** 고품질 크레딧 인증으로 프리미엄 가격 확보

### 10.3 최종 판단

| 질문 | 답변 |
|------|------|
| 현행 방법론으로 K-ETS 외부사업 등록이 가능한가? | **❌ 불가능** — POPLE 방법론은 K-ETS에서 인정되지 않음 |
| 방법론 재개발 후 외부사업 등록이 현실적인가? | **△ 조건부 가능** — 따릉이 선례가 있으나, 민간 플랫폼의 추가성 입증이 최대 허들 |
| 현 시점에서 가장 현실적인 경로는? | **POPLE 유지 + K-VCM 준비** — 한국형 자발적 탄소시장이 출범하면 크레딧 유통 가능성 확대 |
| 장기적으로 가장 가치있는 경로는? | **Verra VCS 또는 Gold Standard** — 글로벌 수요 접근 + CORSIA 연계 가능 |
