# 투트랙 전략: 한국 STO + 해외 블록체인 토큰

> 작성일: 2026-03-10
> 대상: 라이트브라더스 탄소저감 토큰증권 사업

---

## 1. 전략 개요

한국에서는 토큰증권(STO)으로 규제 안에서 상장하고, 해외에서는 글로벌 앱 런칭과 함께 블록체인 유틸리티 토큰을 병행하는 투트랙 전략.

**핵심 원리**: 같은 탄소저감 데이터를 각 시장에 맞는 형태(STO/토큰)로 제공

## 3. Track 1 vs Track 2 비교

| 항목 | Track 1: 한국 STO | Track 2: 해외 블록체인 |
|------|:---:|:---:|
| **대상 시장** | 한국 투자자/기업 | 글로벌 사용자/투자자 |
| **법적 근거** | 자본시장법 (2026.01 통과) | 국가별 상이 (회색지대) |
| **블록체인 개발** | 불필요 (증권사 담당) | **직접 개발 필요** |
| **토큰 성격** | 증권형 (투자 수익) | 유틸리티 (서비스 이용) |
| **거래소** | NXT / KDX (인가 거래소) | Uniswap, 또는 글로벌 CEX |
| **초기 비용** | 중간 (증권사 수수료) | 높음 (개발 + 감사 + 법률) |
| **시장 진입 속도** | 2026 하반기 가능 | 12~18개월 |
| **투자자 신뢰도** | 높음 (규제 내) | 중간 (프로젝트 신뢰에 의존) |
| **유동성** | 초기에 제한적 | 글로벌 유동성 접근 가능 |
| **수익 잠재력** | 안정적 | 폭발적 (but 리스크도 큼) |

## 4. 투트랙 전략의 장점

### 4.1 리스크 분산

- 한국 STO만 할 경우 → 한국 시장 규모에 한정 (자전거 인구 ~1,300만)
- 해외 토큰만 할 경우 → SEC/MiCA 등 규제 리스크 직격, Flowcarbon처럼 하루아침에 무너질 수 있음
- **투트랙** → 한쪽이 흔들려도 다른 쪽이 버팀목

### 4.2 Sweatcoin 선례 (가장 유사한 성공 모델)

| Sweatcoin 전략 | 라이트브라더스 적용 |
|---|---|
| 영국에서 FCA 규제 준수 (앱) | 한국에서 금융위 규제 준수 (STO) |
| SWEAT 토큰은 별도 법인(스위스)에서 발행 | 해외 토큰은 별도 법인(싱가포르/두바이)에서 발행 |
| 앱 1.4억 유저 → 토큰 생태계 유입 | 한국 유저 기반 → 글로벌 확장 |
| Near Protocol 블록체인 사용 | Polygon/Base 등 L2 체인 사용 가능 |

### 4.3 STO 실적이 해외 토큰의 신뢰도가 됨

"한국 금융당국이 인가한 거래소에 상장된 프로젝트"
- 해외 투자자에게 강력한 신뢰 시그널
- ICO 사기 프로젝트와 차별화
- VC/투자자 미팅에서 핵심 셀링포인트

## 5. 한국 STO에서 블록체인 코드가 불필요한 이유

### 5.1 역할 분담 구조

```
┌──────────────┬──────────────────────────────────────┐
│ 라이트브라더스  │  ① SSP 적립/차감 시스템 (기존 코드)     │
│ (발행인)      │  ② SSP → 토큰 전환 요청 API           │
│              │  ③ 탄소저감 데이터 제공                  │
│              │  ★ 블록체인 코드 불필요                  │
├──────────────┼──────────────────────────────────────┤
│ 증권사        │  ① 토큰 발행/관리 (스마트컨트랙트)       │
│ (발행대행)    │  ② 투자자 계좌 관리                    │
│              │  ③ KYC/AML 처리                       │
│              │  ★ 블록체인 인프라 운영                  │
├──────────────┼──────────────────────────────────────┤
│ 거래소        │  ① 토큰 매매 체결                     │
│ (NXT/KDX)   │  ② 분산원장 기록                      │
│              │  ③ 정산/결제                           │
│              │  ★ 블록체인 네트워크 자체를 운영          │
└──────────────┴──────────────────────────────────────┘
```

### 5.2 라이트브라더스가 실제로 개발해야 할 것 (STO)

| 구분 | 내용 | 기술 | 난이도 |
|------|------|------|--------|
| API 연동 | 증권사 토큰 발행 API 호출 | REST API (기존 Node.js) | 낮음 |
| 전환 로직 | SSP → 토큰 전환 비율 계산 | 비즈니스 로직 (JavaScript) | 중간 |
| 데이터 제공 | 탄소저감 증빙 데이터 전송 | JSON/CSV export | 낮음 |
| 지갑 연동 | 사용자 토큰 잔액 조회/표시 | 증권사 API 호출 | 낮음 |
| 이중지급 방지 | SSP 차감 ↔ 토큰 발행 동기화 | DB 트랜잭션 (기존 패턴) | 중간 |

## 6. 해외 블록체인 토큰 구현 시 필요한 것

| 구분 | 내용 | 예상 비용 |
|------|------|----------|
| **스마트컨트랙트** | ERC-20 토큰 (Solidity) | 개발 3,000~5,000만 원 |
| **보안 감사** | 코드 감사 (CertiK, Trail of Bits 등) | 5,000만~2억 원 |
| **법인 설립** | 싱가포르 또는 두바이 (토큰 발행용) | 2,000~5,000만 원 |
| **법률 자문** | 각국 증권법 검토 (미국 Howey Test 등) | 3,000만~1억 원 |
| **DEX 유동성** | 초기 유동성 풀 공급 | 1~5억 원 |
| **블록체인 개발자** | Solidity + Web3 전문 인력 | 연봉 8,000만~1.5억 원 |

> **참고**: 스마트컨트랙트 자체는 ERC-20 표준으로 코드량이 많지 않지만(200~500줄), 보안 감사 비용이 개발비의 3~10배. 해킹 한 번이면 프로젝트가 끝나기 때문.

## 7. 추천 실행 로드맵

### Phase 1: 한국 STO 집중 (2026 상반기~하반기)

- 증권사 파트너 확보
- SSP → 토큰 전환 API 개발
- NXT/KDX 상장
- 첫 거래 실적 확보

### Phase 2: 해외 토큰 준비 (2027)

- 해외 법인 설립 (싱가포르/두바이)
- 스마트컨트랙트 개발 (ERC-20)
- 보안 감사
- 토큰이코노미 설계

### Phase 3: 글로벌 런칭 (2027 하반기~)

- 앱 다국어 지원
- DEX 상장 (Uniswap 등)
- 글로벌 마케팅
- B2B 제휴 (해외 ESG 기업)

**Phase 1을 먼저 하는 이유:**
1. STO는 기존 코드에 API 연동만 추가하면 되어 빠름
2. 한국 실적이 해외 진출의 신뢰 자산이 됨
3. STO 수익으로 해외 토큰 개발 비용 충당
4. 블록체인 개발자 채용/학습 시간 확보

## 8. 리스크 매트릭스

| 리스크 | 심각도 | 대응 방안 |
|--------|:---:|------|
| **한국 STO ↔ 해외 토큰 가치 연동** | 높음 | 같은 탄소저감 데이터에 두 가지 가격이 생기면 차익거래 발생 → 전환 비율 고정 or 단방향 전환만 허용 |
| **이중 발행 (Double Counting)** | 높음 | 같은 SSP로 STO + 해외토큰 동시 전환 방지 → "STO 전환된 SSP"와 "토큰 전환된 SSP" 완전 분리 |
| **미국 SEC 규제** | 높음 | 미국 거주자 참여 차단 (Geo-blocking) + 법률 자문 |
| **EU MiCA 규제** | 중간 | 2024년부터 시행 중, 백서 제출 의무 → 법률 자문 필요 |
| **한국 법인의 해외 토큰 관여도** | 높음 | 한국 법인은 "데이터 제공"만, 토큰 발행은 해외 법인 → 법적 분리 명확히 |

| 리스크 | 시장 제한 | 규제 불확실 | **분산** |
| 성장성 | 제한적 | 높음 | **높음** |
| 실행 가능성 | 높음 | 중간 | **높음** (순차 실행) |
| 신뢰도 | 높음 | 낮음 | **높음** (STO 실적 활용) |
| 총 비용 | 5~10억 | 10~20억 | **15~25억** (순차 집행) |

## 10. 핵심 파트너십 구조 (투트랙)

```
[Track 1: 한국]
  라이트브라더스 → 증권사 → NXT/KDX → 한국 투자자/ESG 기업

[Track 2: 해외]
  라이트브라더스 → 해외 법인 → 스마트컨트랙트 → DEX/CEX → 글로벌 사용자

[공통 기반]
  탄소저감 데이터 엔진 (라이딩 + 중고거래)
  → 환경부/KCCI/Verra 인증
  → SSP 시스템 (T_MEMBER_SSP)
```
