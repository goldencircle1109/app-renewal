# 라이트브라더스 앱 리뉴얼 사업계획서 (BRD)
## Business Requirements Document

**문서작성일:** 2026년 3월 28일
**프로젝트명:** Wright Brothers App Renewal
**담당:** 라이트브라더스 경영진 + 개발팀
**상태:** 최종 승인

## 1. 경영 요약 (Executive Summary)

### 1.1 프로젝트 개요

**라이트브라더스 앱 리뉴얼**은 기존 레거시 시스템을 완전히 현대화하여, **교통수단 자동 감지 기능**과 **탄소 감축량 자동 계산**, 그리고 **멀티스포츠 도전 플랫폼**으로 진화시키는 전략적 프로젝트입니다.

현재의 자전거 전용 앱에서 **다중 스포츠(러닝, 사이클링, 하이킹)와 친환경 교통 기반의 종합 활동 추적 및 보상 플랫폼**으로 탈바꿈합니다.

### 1.2 리뉴얼 필요성

#### 기술부채 해결
- 현재 시스템: Node.js + Vue.js 혼재, TypeScript 미사용, 테스트 없음 (11만+ 줄 코드)
- 보안 취약점: API 키, SMS 키 하드코딩, 커스텀 토큰 (JWT 아님) → 즉시 해결 필수
- 유지보수 불가능: Spring Boot 7개 프로젝트 건드릴 개발자 없음

#### 시장 기회 확대
- **한국 러닝 인구 1,000만 명**, 러닝화 시장 1조 원 규모 성장 중
- Strava, Nike Run Club, 런데이 등 경쟁 앱들이 한국 로컬 기능 부족 (크루 관리, 정기 런 이벤트 없음)
- **파크런** 모델(호주, 글로벌 1,100만 등록)이 한국 미진출 → 시장 공백
- 교통수단 대체 탄소 감축이 **ESG/그린뉴딜** 정부 정책과 부합

#### 수익 다각화
- Phase 1: SSP(포인트) 기반 사용자 참여 증대
- Phase 2: 파크런, 구간 랭킹 프리미엄 구독 (월 5,900원)
- Phase 3: 깜깜이방 커머스 (B2B 대리점 납품 + 재판매) + 일반 전자상거래

### 1.3 전략 요약

```
현재 (WB3)              →  Phase 1 (14주)           →  Phase 2 (18주)         →  Phase 3 (18주)
─────────────────────────────────────────────────────────────────────────────────────────
자전거 전용            →  3Way Sensor            →  멀티스포츠             →  커머스
GPS 기록 앱            →  자동 교통수단 감지      →  운영 이벤트 플랫폼     →  B2B/B2C 판매
                      →  탄소 감축 계산         →  크루 커뮤니티         →  C2C 마켓플레이스
                      →  SSP 보상 시스템       →  파크런/세그먼트 랭킹  →  렌탈 시스템
```

**총 개발 기간:** 약 13개월 (정상), 약 10-11개월 (가속화)

## 2. 제품 비전 및 전략 (Product Vision & Strategy)

### 2.1 미션 선언문

> **"자동차 대신 내 발과 자전거로 이동하는 모든 순간, 탄소를 줄이고 보상받으며, 함께 달린다."**

라이트브라더스는 개인의 친환경 이동 선택을 **자동으로 감지**하고, **사회적 영향력을 수량화**하며, **커뮤니티와 연결**하는 플랫폼입니다.

### 2.2 3년 제품 비전 (2026~2029)

#### 연도별 진화 로드맵

**2026년 (원년): 활동 추적 & 탄소 기반 구축**
- 3Way Sensor (자동 교통수단 감지)
- 탄소 감축량 자동 계산 및 시각화
- SSP 포인트 보상 시스템
- 타겟: 활성 사용자 5만 명, 월간 탄소 감축 100톤

**2027년 (성장): 멀티스포츠 & 커뮤니티 확대**
- 러닝/사이클링/하이킹 멀티스포츠 지원
- 파크런 한국 신규 런 이벤트 (주간 정기 5K)
- Strava 세그먼트 랭킹 시스템 (한국 주요 코스)
- 크루 기반 커뮤니티 및 대항전
- 타겟: 활성 사용자 30만 명, 월간 탄소 감축 500톤

**2028~2029년 (통합): 생태계 & 커머스 완성**
- 깜깜이방 커머스 (B2B 대리점 특정매입)
- 전자상거래 일반화 + AI 상품 상세페이지 자동 생성
- 자전거/러닝용품 렌탈 시스템
- SSP → 외부 상품 교환 (Naver Pay, Kakao Gift, Onnuri)
- 탄소 인증서 발급 및 거래
- 타겟: 활성 사용자 100만 명, 월간 탄소 감축 1,000톤, 연간 GMV 100억 원

### 2.3 전략적 포지셔닝: 경쟁 우위

#### 라이트브라더스 ≠ 다른 앱들

| 경쟁사 | 강점 | 약점 (우리 기회) |
|--------|------|------------------|
| **Strava** | 구간 랭킹, 고급 데이터 | 유료 많음, 한국 로컬 기능 없음, 커뮤니티 약함 |
| **Nike Run Club** | 브랜드, 대중성 | 로컬 크루 기능 없음, 이벤트 없음 |
| **가민 커넥트** | 기기 연동, 정밀 데이터 | 가민 디바이스 필수 |
| **런데이** | 초보자 친화 | 커뮤니티/대회 기능 제한, 데이터 심화 기능 없음 |

#### 라이트브라더스의 독점적 강점

1. **3Way Sensor: 자동 교통수단 감지**
   - 자동차/택시 vs 버스/지하철 vs 도보/자전거 **자동 구분**
   - Wi-Fi SSID + 지오펜스 + GPS 속도 조합 (국내 특화)
   - 사용자는 버튼 누를 필요 없음 (완전 자동)

2. **탄소 기반 보상 시스템**
   - 이동 거리만이 아니라 **CO₂ 절감량 기반** SSP 보상
   - T-map API로 자동차 대비 기준 거리 자동 계산
   - MRV(검증 가능한 감축량) 입증 가능 → 탄소시장 확대 가능

3. **파크런 + 세그먼트 랭킹의 결합**
   - 파크런: 호주 모델, 주간 무료 정기 러닝 이벤트 (한국 미진출)
   - 세그먼트 랭킹: Strava 같은 구간 경쟁 + 한국 러닝 코스 특화
   - 이 둘의 결합 = **크루 기반 정기 이벤트 + 개인 기록 경쟁** (경쟁 앱 없음)

4. **커뮤니티-중심 설계**
   - 크루 생성/관리 → 크루별 런 일정 → 크루 간 대항전 (완전 탑-다운 기능)
   - Strava는 개인 중심, NRC는 브랜드 중심 → 우리는 **로컬 러닝 크루 문화 완벽 지원**

5. **멀티스포츠 + GPX-in 자동 과정 생성**
   - 한국관광공사, 지방자치단체, 국립공원이 GPX 업로드 → **관리자 없이 자동 과정 생성**
   - 러닝, 사이클링, 하이킹 한 플랫폼에서 통합 관리
   - 경쟁사: 각 스포츠별 별도 앱 필요

#### 복제 불가능한 경쟁 해자 (Moat)

1. **기술 해자: 3Way Sensor**
   - 한국 교통 환경에 특화된 Wi-Fi SSID + 지오펜스 조합
   - 2년 이상의 패턴 학습 데이터 축적 → 정확도 계속 개선
   - 경쟁사가 복제하려면 최소 1년 개발 + 검증 필요

2. **사용자 기반 해자: 크루 네트워크 효과**
   - 크루에 50명 이상 모이면, 더 이상 이전할 수 없음 (커뮤니티 텍스트 락인)
   - 크루별 활동 기록, 대항전 결과가 축적되면 이전 불가능
   - 1년 뒤 5만 명 사용자 → 각 도시마다 500명 크루 형성

3. **데이터 해자: 탄소 감축 실적**
   - 3년간 누적된 100만 명 × 3년 = 300만 건의 활동 기록
   - 교통 선택 패턴, 지역별 탄소 감축 인사이트 보유
   - 정부, KTO, 기업 ESG 보고에 활용 → 유상 데이터 서비스 가능

4. **정부/기관 관계 해자: 공식 파크런 + 관광지 과정**
   - KTO, 지방자치단체와 파트너십 → 공식 채널 독점
   - 경쟁사는 비공식 사용자 커뮤니티 수준

## 3. 타겟 사용자 (Target Users)

### 3.1 주요 사용자 페르소나

#### 페르소나 1: 탄소 감축형 통근자 (Carbon-Conscious Commuter)

**프로필**
- 연령: 28~45세, 직장인, 아파트 거주
- 거주지: 수도권 또는 광역시 (대중교통 발달)
- 월소득: 300~500만원 (중산층)
- 스포츠: 자전거 또는 도보 통근 (5~15km), 주말 러닝 취미

**동기**
- 자동차 연료비 절감 (유류비 + 주차비)
- 환경 의식 (탄소 발자국 감소)
- 건강 (일일 운동 습관)
- 사회적 임팩트 (남의 눈에 띄는 ESG 활동)

**페인포인트**
- 통근 기간이 오래되면 몸과 마음이 지침 (동기 부족)
- 얼마나 탄소를 줄였는지 시각적으로 보고 싶음
- 혼자가 아닌 같은 생각의 사람들과 함께하고 싶음
- 정부나 기업에 "환경 활동 실적"을 인증받고 싶음

**우리 플랫폼에서의 행동**
- 매일 아침 집→지하철역→회사 이동 기록
- SSP 포인트 자동 적립 (버튼 누를 필요 없음)
- 주간 탄소 감축량 대시보드 확인 ("이번 주 280kg CO₂ 절감" 시각화)
- 월간 리포트 SNS 공유 ("라이트브라더스에서 자동차 대신 대중교통으로 500kg 탄소 절감!" + 인증서)
- 회사 ESG 활동 증명 자료로 사용

**기대 시장규모**
- 수도권 자전거/도보 통근자: 약 100만 명 중 10% 타겟 = 10만 명
- 전국 확대 시 300만 명

**1년차 목표**
- 활성 사용자 1만 명
- 월간 활성 사용자(MAU) 3,000명
- 일일 사용 시간: 5분 (아침 러닝/자전거 + 저녁 리뷰)

---

#### 페르소나 2: 도전형 러너/사이클리스트 (Challenge-Seeking Runner/Cyclist)

**프로필**
- 연령: 25~50세, 다양한 직업군 (직장인 + 자유직)
- 거주지: 전국 (온라인 커뮤니티 중심)
- 소비력: 상대적으로 높음 (월소비 100만원 이상 운동용품)
- 스포츠: 마라톤, 자전거 라이딩, 트레일 러닝 (주 3~5회, 거리 30~100km/주)

**동기**
- **개인 기록(PR) 향상**: 같은 코스에서 시간 단축 경쟁
- **랭킹 욕구**: 구간별 "코스 킹" 또는 "단골 러너" 인정
- **커뮤니티 소속감**: 같은 실력대의 동호인들과 경쟁 및 공유
- **배지/인증**: 100k 완주, 산 정상 10개 모두 제패 등 시각적 성취

**페인포인트**
- Strava는 가격이 비싸고 프리미엄 기능이 많음 (월 1.99$+)
- Nike Run Club은 한국 러닝 크루 기능이 없음
- 기존 앱들은 "주간 정기 무료 러닝 이벤트"가 없음
- 같은 크루 사람들과의 그룹 챌린지 기능 부족

**우리 플랫폼에서의 행동**
- 매주 토요일 "WB 파크런" 참가 (수도권 여의도, 올림픽공원 등 5km)
- 여의도 한강 "구간 랭킹" 개인 기록 계속 갱신 시도
- 자신이 속한 "마라톤 크루" 앱에서 주간 대항전 참가
- 프리미엄 구독 (월 5,900원): 라이브 세그먼트 (달리는 중 실시간 PR 표시)
- SNS에 기록 공유 ("라이트브라더스 세그먼트 킹 달성!")

**기대 시장규모**
- 한국 마라톤 참가자: 연 100만 명 (약 400개 대회)
- 정기적 러닝/사이클링 동호인: 약 50만 명
- 우리 타겟 (월 활동 2회 이상): 약 30만 명 중 10% = 3만 명

**1년차 목표**
- 활성 사용자 2만 명
- 월간 활성 사용자(MAU) 8,000명
- 프리미엄 전환율: 15% (약 1,200명)
- 월간 프리미엄 구독 매출: 약 600만원

#### 페르소나 3: 크루 리더/커뮤니티 빌더 (Crew Leader / Community Builder)

**프로필**
- 연령: 30~55세, 스포츠 매니아, 자신의 "커뮤니티" 주인 의식
- 거주지: 대도시 또는 지역 중심도시 (지역 러닝/자전거 크루 주최)
- 특성: 사교성 높음, 온라인 커뮤니티 경험 있음 (카페, 밴드, 페이스북 그룹)
- 스포츠: 본인도 활발 (주 3~5회) + 크루 운영 경험 (5~50명 규모)

**동기**
- **크루 일관성 강화**: 산발적 카톡 대신, 앱 기반 일정/기록 관리
- **크루 성장**: "라이트브라더스 크루" 탭에서 신입원 자동 모집
- **크루 명예**: 크루별 누적 거리, 평균 페이스, 크루 랭킹 시각화
- **이벤트 운영 단순화**: 주간 런 일정, 참가 인원, 완주 기록을 자동 집계

**페인포인트**
- 현재: "매주 카톡 그룹에서 일정 공지 → 캘린더 링크 공유 → 끝난 후 기록 수동 정리"
- 기존 앱: 크루 기능이 없거나, 있어도 너무 단순함 (카톡이 더 나음)
- 실적 인증: "우리 크루가 이번 달 탄소 500kg 절감했어" 는 증명 불가능

**우리 플랫폼에서의 행동**
- "성수 자전거 크루" 생성 (50명 멤버)
- 매주 토요일 04:00 한강 코스 산악자전거 라이드 일정 등록

- "성수 자전거 크루" 월간 리더보드: 누가 가장 자주, 가장 멀리 탔는가?
- "OO크루 vs XX크루" 월간 대항전 → "성수 크루 이겼다!" 자동 집계 및 알림

**기대 시장규모**

**1년차 목표**

#### 페르소나 4: 스포츠용품 소매점주/딜러 (Sports Retail Dealer)

**프로필**
- 연령: 40~65세, 자전거/러닝용품 전문점 운영 (10~30명 직원)
- 사업 규모: 월매출 3,000~8,000만원
- 고통: **시즈 지난 재고 처리** (정가 판매 불가, 수입사 계약 해지 위협)
- 시스템: 기존 오프라인 판매 + 카톡/카페 중심 온라인

**동기**
- **재고 현금화**: 판매하지 못한 지난 시즈 상품을 몰래 할인하고 싶지만, 공개적으로 할인하면 수입사에서 계약 해지
- **신용도 유지**: "라이트브라더스"라는 브랜드 이름으로만 판매 (자신의 이름이 안 노출)
- **새 시즌 자금**: 재고 현금화 → 새 시즌 상품 매입 자금 확보

**우리 플랫폼에서의 행동 (Phase 3)**
- "깜깜이방(Dark Commerce)" 참가
- 지난해 로드바이크 정가 300만원 → 구입가 210만원으로 라이트브라더스에 납품
- 라이트브라더스가 240만원에 소비자에게 판매 (판매자 표시는 "라이트브라더스" only)
- 딜러가 받는 금액: 210만원
- 라이트브라더스 수수료: 30만원 (마진 12.5%)
- 결과: 딜러는 현금 210만원 확보 + 수입사에는 "누가 할인했는지" 알 수 없음

**기대 시장규모**
- 자전거 전문점: 2,000~3,000개 (전국)
- 러닝용품 매장: 500~800개
- 데드스톡(재고 부담): 약 2,000~3,000억원/년
- 우리 타겟 (거래 가능한 점포): 1년차 30개, 3년차 500개

**1년차 목표**
- 참여 딜러: 30개
- 월간 등록 상품: 150개
- 월간 판매 상품: 105개 (70% 전환율)
- 월간 GMV(거래액): 5,250만원
- 월간 수수료 수익: 630만원

## 5. 경쟁 분석 (Competitive Analysis)

### 5.1 직접 경쟁사 (Direct Competitors)

#### 1) **Strava**

**회사:** Strava Inc. (미국, 2009년 설립)
**사용자:** 글로벌 7,000만 명, 한국 약 50만 명
**주요 기능:** GPS 기록, 세그먼트 랭킹, 소셜 공유, 라우트 검색
**비즈니스 모델:** 프리미엄 구독(월 $2.99), 광고, 스폰서십

**강점:**
- 글로벌 브랜드 인지도 높음
- 세그먼트 랭킹 시스템 잘 정립 (전 세계 수백만 개 세그먼트)
- 사이클리스트/러너 모두 포함한 멀티스포츠
- 활발한 커뮤니티 (팔로우, 쿠도스, 댓글)

**약점:**
- **한국 로컬 기능 부족**: 파크런 같은 정기 이벤트 없음
- **크루 관리 기능 빈약**: 카톡이 더 효율적 (한국 사용자 기준)
- **탄소 감축 관련 기능 없음**: 환경 이슈에 대응 미흡
- **비싼 구독료**: 한국 사용자는 월 4,900원 선택 부담
- **커뮤니티 기반 경쟁 약함**: 개인 기록 경쟁 중심

**우리의 경쟁 전략:**
- 파크런 + 세그먼트 결합 (Strava는 세그먼트만)
- 크루 기반 이벤트 (Strava는 개인 중심)
- 탄소 감축 보상 (Strava는 없음)
- 프리미엄 가격 낮춤 (월 5,900원 vs Strava 월 4,900원)

### 5.4 경쟁 해자 (Competitive Moat): 복제 불가능한 강점

#### 1. **기술 해자: 3Way Sensor**

- **버스/지하철 자동 감지**: Wi-Fi SSID 패턴 DB (서울 버스, 지하철, KTX 특화)
- **경쟁사 진입 장벽**: 2년 이상 국내 데이터 수집 + 정확도 검증 필요
- **패턴 학습 우위**: 100만 명 × 2년 = 200만 건의 활동 데이터 → 정확도 계속 개선

#### 2. **사용자 기반 해자: 크루 네트워크 효과**

- **전환 비용 (Switching Cost) 높음**: 크루에 50명 모이고 3개월 활동하면, 다른 앱으로 이전 불가능 (커뮤니티 텍스트 록인)
- **한국 문화 특성**: 한국 러닝/자전거 크루는 "**정기 런**" 중심 → 앱 없이 운영 불가능
- **경쟁사 극복 어려움**: Strava가 크루 기능을 만들어도, 기존 500개 크루는 우리 플랫폼에 이미 정착

#### 3. **데이터 해자: 탄소 감축 실적**

- **독점 데이터**: 3년 누적 100만 명 × 3년 = 300만 건 활동 기록 → 지역별/시간대별 탄소 감축 인사이트
- **정부/기관 가치**: KTO, 지자체, 환경부가 우리의 데이터 → "한국 국민 탄소 감축 실적 보고서" 제작 (유상 데이터 서비스)
- **경쟁사 따라 불가**: 데이터는 3년 축적 필수 (즉시 복제 불가)

#### 4. **파트너십 해자: 정부/기관 공식 채널**

- **KTO (한국관광공사)**: 파크런 공식 한국 채널 (독점 협력)
- **지방자치단체**: "서울시 공식 탄소 감축 앱" 지위
- **경쟁사 극복 어려움**: 정부 협력은 신뢰 기반 → 선발자 우위

#### 5. **기술 진입 장벽: GPX 자동 과정 생성**

- **자동화 수준**: GPX 업로드 → 시스템이 5분 내 과정/체크포인트/도전/리더보드 자동 생성
- **경쟁사 따라 어려움**: 개발에 6개월+ 필요
- **제휴처 확보 우위**: KTO, 지자체가 GPX 수동 등록은 번거로움 → 우리 자동화 선호

#### 6.1.2 탄소 크레딧 잠재 수익

| 시나리오 | 활성 사용자 | 연간 감축량 추정 | 크레딧 수익 (1만원/톤) |

#### 6.1.3 SSP 포인트 에코시스템 설계

```
[사용자 활동]
   탄소 감축 × SSP율(100SSP/kg CO₂)
   + 활동 기반 적립
     - 걷기: 5 SSP/km
     - 달리기: 8 SSP/km
     - 자전거: 10 SSP/km
     - 대중교통: 3 SSP/회
          ↓
[SSP 생태계]
   Phase 1: 앱 내 혜택 (챌린지 참여권, 배너 노출)
   Phase 2: 외부 포인트 교환 (네이버페이, 카카오 선물하기, 온누리상품권)
   Phase 3: 커머스 캐시 활용 (깜깜이방 구매 할인)
```

### 6.1.5 MRV 증거 체계: 자동차 소유/이용 증빙 시스템

탄소배출권 전문가 요구에 따라, "자동차 대신 친환경 수단으로 전환했다"를 증빙하기 위한 3단계 증거 체계를 구축한다.

**3-Tier Evidence System:**

| Tier | 대상 | 방식 | 증거 |
|------|------|------|------|
| **Tier 1: 자동 검증** | 본인 명의 차량 | API 자동 | 운전면허 API + 자동차등록원부 + 보험가입경력 |
| **Tier 2: AI 반자동** | 가족차/법인차/리스/렌탈 | API + 서류 + AI OCR | 운전면허 API + Claude Vision OCR (보험증권, 계약서 등) |
| **Tier 3: 행동 기반** | 서류 없는 경우 | API + 센서 | 운전면허 API + 하이패스 이력 + 3Way Sensor Vehicle 감지 이력 |

**핵심 API 연동:**
- 도로교통공단 운전면허 자동검증 시스템 (dlv.koroad.or.kr) — 면허 진위/종류/취득일
- CODEF API — 자동차등록원부 갑부(소유 이력), 보험다모아(보험가입경력), 하이패스 이용내역
- Claude Vision API — 서류 사진 자동 OCR + 검증 (보험증권, 리스계약서, 재직증명서 등)

**AI OCR 판정 결과:** AUTO_APPROVED(자동승인) / PARTIAL_MATCH(간편리뷰) / MANUAL_REVIEW(직접확인) / REJECTED(반려)

**비용:** 월 ~45-80만원 (CODEF + 면허API + Claude Vision) vs 수동 검수 인력 200만+/월 → 60-75% 절감

> 상세 리서치: `3way sensor/06_자동차소유_MRV증거체계_리서치.md`

#### 6.3.1 커머스 수수료 구조 (깜깜이방)

딜러 익명성 보장 + 특정매입 구조를 전제로 한 수수료율 설계. 딜러가 재고 소진의 가치와 신원 보호의 편익을 누리는 대가로 시장 평균(네이버 스마트스토어 2~5.85%)보다 높은 수수료를 수용한다.

| 카테고리 | 수수료율 | 근거 |
|---------|---------|------|
| 완성차 로드/MTB (200만원 이상) | 8~10% | 고가 제품, 마진 폭 15~25% |
| 완성차 생활자전거 (100만원 미만) | 12~15% | 마진 폭 25~35%, 회전율 높음 |
| 고급 부품/파츠 (시마노 등) | 10~12% | 마진 폭 20~30% |
| 일반 부품/소모품 | 12~15% | 마진 폭 25~35% |
| 헬멧/의류/악세서리 | 15~20% | 마진 폭 30~50% |
| 프리미엄 러닝화 (10만원 이상) | 12~15% | 마진 폭 30~40% |
| 러닝 용품/기타 | 15~18% | 마진 폭 35~45% |
| C2C 중고 거래 | 3.5% | 기존 WB3 수준 유지 |

**수수료 산정 공식:**
```
소비자 판매가 = 딜러 Floor Price ÷ (1 - 수수료율)
딜러 정산액 = 소비자 판매가 - (소비자 판매가 × 수수료율)

예시 (완성차 300만원 정가):
  딜러 Floor Price: 220만원 (딜러 마진 최소 보전선)
  수수료율: 10%
  소비자 판매가: 220만 ÷ 0.9 = 244만원 (정가 대비 18.7% 할인)
  WB 수수료 수익: 24.4만원
  딜러 정산: 219.6만원
```

#### 6.3.2 딜러 티어별 정산 사이클

| 티어 | 조건 | 정산 사이클 | 혜택 |
|------|------|-----------|------|
| 브론즈 | 신규 등록 | 구매확정 후 5영업일 | 기본 수수료 |
| 실버 | 가입 3개월 + 거래 10건 이상 | 구매확정 후 3영업일 | 수수료 0.5% 할인 |
| 골드 | 가입 6개월 + 거래 30건 이상 | 구매확정 후 1영업일 | 수수료 1% 할인 + 우선 노출 |

> 구매확정: 배송 완료 후 7일 경과 + 반품 없음 = 자동 확정 처리

```

손익분기 GMV = 고정비 160만원 ÷ 수수료율 12%
             = 약 1,333만원/월 GMV

손익분기 거래 건수 (평균 단가 150만원 가정):
  = 1,333만원 ÷ 150만원 = 약 9건/월

설계 문서 기재 손익분기 GMV: 약 4,000만원/월 (~80건, 평균단가 50만원)
→ 보수적으로 4,000만원/월 GMV 달성을 목표로 설정
```

## 11. Phase 4: 탄소 토큰증권 (STO) + 글로벌 확장

### 11.1 전략 개요

SSP 포인트와 탄소저감 데이터를 토큰증권(STO)으로 발행하여 국내 자본시장에 진입하고, 해외 블록체인 유틸리티 토큰으로 글로벌 시장에 진출하는 투 트랙 전략.

### 11.2 투 트랙 전략

| 항목 | Track 1: 한국 STO | Track 2: 해외 블록체인 |
|------|-------------------|---------------------|
| 법적 근거 | 자본시장법 + 전자증권법 (2026.01.15 시행) | 싱가포르/두바이 법인 |
| 거래소 | NXT 또는 KDX | Uniswap/SushiSwap (DEX) |
| 토큰 유형 | 증권형 토큰 (투자 목적) | 유틸리티 토큰 (ERC-20) |
| 라이트브라더스 역할 | SSP 잠금/해제 API + 탄소 증빙 데이터 | 별도 법인 + 스마트컨트랙트 |
| 투자자 신뢰 | 높음 (규제 하) | 프로젝트 실적에 의존 |
| 유동성 | 초기 국내 한정 | 글로벌 |
| 타겟 런칭 | 2027 Q1-Q2 | 2027 Q3-Q4 |

### 11.3 수익 구조

| 수익원 | 예상 연 수익 |
|--------|------------|
| STO 거래 수수료 분배 | 1-5억원 |
| B2B 탄소크레딧 판매 (기업 ESG) | 5-20억원 |
| 프리미엄 구독 부스트 (토큰 가속) | 1-3억원 |
| 해외 토큰 생태계 (DEX 수수료, 스테이킹) | $1-10M |
| 브랜드 파트너십 (ESG 기업 후원) | 1-10억원 |

### 11.4 비용 추정

| 항목 | 비용 |
|------|------|
| 증권법 법무 (STO 투자설명서) | 3,000만-1억원 |
| 탄소 인증 (KCCI/제3자 감사) | 2,000만-5,000만원 |
| 스마트컨트랙트 개발 + 보안 감사 | 8,000만-2.5억원 |
| 해외 법인 설립 + 법률 자문 | 4,000만-1.3억원 |
| DEX 초기 유동성 | 5,000만-2억원 |
| **합계** | **약 2.3-10.6억원** |

### 11.5 핵심 리스크

| 리스크 | 대응 |
|--------|------|
| FSC 승인 거부 | 사전 컨설팅 + 규제 샌드박스 참여 |

### 12.5 외국인 관광객 시나리오

```
외국인이 한국 방문 → 앱 다운로드 → Google/Apple 가입 (30초)
→ 언어 선택 (English) → 관광 모드 (집/회사 등록 스킵)
→ 대표코스 60선에서 자전거/하이킹 → GPS 매칭 → 스탬프 수집 → SSP 적립
→ SSP로 관광지 QR 쿠폰 교환 or Phase 4 토큰 전환
→ "Completed Jeju Coastal Ride!" SNS 공유 → 한국 자전거 관광 글로벌 홍보
```

### 12.6 관광공사 연계 모델

관광공사가 외국인 관광객의 SSP를 대행하여 온누리상품권/관광 쿠폰으로 교환:
- 관광공사 ESG 예산으로 온누리상품권 구매
- 외국인이 SSP 교환 요청 → 관광공사 대납 → 현장 지급
- 외국인은 전통시장에서 소비 → **자전거 관광 + 지역경제 활성화 + ESG** 선순환
