### 2. 📊 완전 GTM 전략 - Part 1
**파일:** `GTM_Strategy_Phase1.md`

상세한 실행 전략 및 전술:

#### 섹션별 목차
- **1. 기존 회원 전환 전략** (13만명 → 2.6만명~3.9만명)
  - 마이그레이션 캠페인: "업그레이드하면 5,000 SSP 보너스"
  - 푸시, 이메일, SNS, 오프라인 QR 코드 활용

- **2. 자전거 커뮤니티 활용** (6,500~9,000명)
  - 동호회 직접 접근: 100개 크루 파트너십
  - 스트라바 한국 사용자 공략 (50만명)
  - 자전거 카페 및 온라인 커뮤니티 전략

- **3. 러닝 크루 시장 공략** (5,000~7,000명)
  - 서울 상위 러닝 크루 200개 확보
  - 마라톤 이벤트 스폰서십 (연 5개 이벤트)
  - NRC 사용자 리타겟팅

- **4. 기업 ESG 채널** (9,400명)
  - 기업 직원 탄소저감 챌린지
  - 지자체 친환경 출퇴근 캠페인
  - 한국관광공사(KTO) 파트너십

- **5. 디지털 마케팅** (2,500~5,000명)
  - 앱스토어 ASO (App Store Optimization)
  - Google Ads & Naver Ads (월 500만원)
  - YouTube, TikTok, Instagram 콘텐츠
  - 인플루언서 시딩 (10명)

- **6. 입소문 메커니즘** (3,600명)
  - 추천 시스템: "친구 초대하면 SSP 2,000"
  - SNS 자동 공유 카드
  - 크루 초대 시스템

- **7. 타임라인 & 예산**
  - 월별 상세 일정 (4월~7월)
  - 채널별 마케팅 예산 (총 13,800만원)
  - 마일스톤 체크포인트

**읽는 시간:** 45분
**대상:** 마케팅 담당자, 제품 매니저

## 🎯 빠른 참조 (Quick Reference)

### 목표 (4개월 후)
```
신규 가입: 14,450명
월간 활성 사용자: 10,000명 (70% 활성율)
일일 활성 사용자: 2,500명
누적 탄소 감축: 300톤
```

### 핵심 메시지
```
"자동차 대신 자전거/도보로 이동하면, 자동으로 탄소를 줄이고 포인트로 보상받습니다."
```

### 6가지 주요 채널 (예상 기여도)
1. **기존 회원 전환** - 22% (3,250명)
2. **자전거 커뮤니티** - 44% (6,500~9,000명)
3. **러닝 크루** - 35% (5,000~7,000명)
4. **기업 ESG + 지자체** - 41% (6,000~8,000명)
5. **디지털 마케팅** (광고, SNS) - 17% (2,500~5,000명)
6. **입소문 (추천)** - 25% (3,600명)

*총합이 100% 초과인 이유: 채널 간 중복 (동일 사용자가 여러 채널을 통해 인식)*

### 마케팅 예산 (4개월 총 13,800만원)
- 기존 회원 전환 (SSP 보너스): 5,500만원
- Google/Naver 광고: 2,300만원
- 오프라인 이벤트 (마라톤, 러닝): 1,400만원
- 인플루언서 시딩: 1,500만원
- 자재 & 기타: 1,900만원
- 자동화 도구: 200만원

### 평균 CAC (고객 획득 비용)
```
총 비용: 13,800만원
신규 가입: 14,450명
CAC: 약 9,500원 ✓ (효율적 - 업계 평균 15,000~20,000원)
```

### ROI (Return on Investment)
```
4개월 마케팅 투자: 13,800만원
Phase 1 예상 수익: 13,653만원
ROI: 약 0.99배 (손익분기점)

주의: Phase 1은 사용자 확보 중심이고,
Phase 2~3에서 프리미엄/커머스 수익이 크게 증가
```

## ⚠️ 주요 위험 & 대응

| 위험 | 확률 | 대응 |
|-----|-----|------|
| 신규 가입 부족 | 40% | 광고비 추가 → 기대 효과 +2,000~3,000명 |
| 앱 품질 문제 | 35% | 론칭 전 QA 강화 → 버그 <10개 목표 |
| 경쟁사 대응 | 25% | 탄소+커뮤니티로 차별화 |
| 리소스 부족 | 50% | 자동화 + 인턴/프리랜서 활용 |

**결론:** 1순위 2개(신규 가입, 앱 품질)에 집중하면 달성 가능

## ✅ 성공의 황금률

```
성공 = 좋은 제품 × 명확한 메세징 × 올바른 채널 × CEO의 시간 투자

조건:
1. 제품 안정성 90% 이상 (버그 <10개)
2. 메시징 일관성 (모든 채널에서 동일)
3. 올바른 채널 선택 (기존+동호회+기업이 80%)
4. CEO의 주 20시간 직접 접촉 (동호회 리더, 기업 담당자)
5. 데이터 기반 조정 (주간 대시보드 리뷰)
```
