# 라이트브라더스 앱 리뉴얼 GTM 전략
## Go-to-Market Strategy for 10,000 MAU Target

**작성일:** 2026년 3월 28일
**목표:** Phase 1 런칭 후 10,000 MAU 달성 (약 4~6개월)
**타겟 시장:** 한국 자전거/러닝 커뮤니티, ESG 의식형 직장인
**마케팅 예산:** 월 500만원 (약 4개월 = 2,000만원)
**담당:** CEO (전략) + 1명 마케팅/커뮤니티 담당자

## 1. 기존 회원 전환 전략 (Existing User Migration)

### 1.1 현황 분석
- **기존 WB3 앱 등록자:** 약 130,000명
- **월간 활성 사용자(MAU):** 추정 8,000~12,000명 (활성율 6~9%)
- **예상 전환 잠재력:** 기존 13만 사용자 중 20~30% = 2.6만~3.9만명
- **목표 달성의 80% 이상이 기존 회원 전환에서 나올 것**

### 1.2 마이그레이션 캠페인: "업그레이드하면 5,000 SSP 보너스"

#### 캠페인명
**"라이트브라더스 신세계 오픈 - 새 앱으로 가면 보너스 5,000 SSP!"**

**1단계: 푸시 알림 (런칭 2주 전)**
- 기존 WB3 앱의 모든 사용자에게 매일 1회 푸시
- 메시지: "새 라이트브라더스가 나왔어요! 자동차 대신 자전거/도보로 이동하면 자동으로 탄소 감축을 측정해드립니다. 지금 다운로드하고 5,000 SSP 보너스를 받으세요!"

**2단계: 이메일 캠페인 (런칭 1주 전, 런칭 당일, 런칭 +1주)**

- **기존 회원 전환율:** 20~25% (2.6만~3.25만명)
- **SSP 보너스 비용:** 130,000명 × 5,000 SSP × 환산가 100원/SSP = 약 6,500만원
  - 분할 지급으로 1차 500만원, 2차 1,500만원, 3차 4,500만원 (월별)
- **신규 가입 유도:** 기존 회원 추천으로 3,000~5,000명 추가

## 2. 자전거 커뮤니티 활용 (Community Leverage)

### 2.1 시장 규모 분석
- **한국 자전거 동호회 수:** 약 3,000~5,000개 (5~500명 규모)
- **활성 멤버:** 약 30만~50만명
- **온라인 커뮤니티:**
  - 스트라바 한국 사용자: 약 50만명
  - 자전거 카페 (네이버): 약 500개, 월 활동 회원 50만명
  - 바이크포럼: 약 30만명 등록

### 2.2 전술 1: 동호회 직접 접근 (Grassroots Outreach)

**목표:** 상위 100개 자전거 동호회 확보

**프로세스:**
1. 자전거 동호회 리스트업 (규모 50명 이상)
   - 서울/경기/부산/대구/인천 중심
   - 카페 멤버 수 + 활성도 기준 상위 100개 선정
2. 동호회 리더 개인 연락 (CEO 직접 이메일 + 카톡)
   ```
   제목: 라이트브라더스 새 앱 론칭 - OO 자전거 크루와 함께하고 싶습니다

   본문:
   안녕하세요. 라이트브라더스 CEO입니다.

   저희 새 앱이 런칭되었고, 한국의 자전거 문화를 가장 잘 이해하는 분들과
   함께하고 싶습니다.

   OO 크루가 라이트브라더스 앱의 공식 파트너 크루가 되면:
   - 크루 전용 뱃지 (OO 크루의 증명)
   - 월간 크루 대항전 자동 개최
   - 크루 최우수 멤버에게 매달 선물 (자전거용품 1만원 상당)
   - 오프라인 이벤트 시 협찬 및 홍보

   관심이 있으시면, 이번 주말 강남역 카페에서 간단한 만남을 가질 수 있을까요?

   감사합니다.
   라이트브라더스 CEO [이름]
   ```
3. 초기 50명 리더 미팅
   - 1차: 강남역 카페 (10명씩, 5회)
   - 2차: CEO가 동호회 야외 라이딩 이벤트에 직접 참석 (5회)
4. 파트너 크루 협약 체결
   - 서명: 간단한 1페이지 협약서 (의무 X, 상호 지원)
   - 크루 이름 등록, 로고 업로드

**대사 인센티브:**
- 크루 리더 1명이 크루 전체를 라이트브라더스로 이동하면:
  - 리더: SSP 50,000 보너스
  - 크루원 1명당: SSP 10,000 보너스 (최대 20명까지 = 20만 SSP)

### 2.3 전술 2: 스트라바 한국 커뮤니티 공략

**현황:**
- 스트라바 한국 사용자: 약 50만명
- 스트라바 세그먼트: 한국 주요 자전거/러닝 코스 2,000~3,000개 등록

**전략:**
1. 스트라바 팔로워 5,000명 이상인 한국 유명 사이클리스트 리스트업 (상위 50명)
2. 스트라바 계정에서 DM 발송:
   ```
   안녕하세요!

   라이트브라더스라는 새로운 앱이 나왔는데,
   여러분처럼 성실하게 기록하는 사이클리스트들을 기대리고 있습니다.

   라이트브라더스의 특징:
   - 스트라바와 동기화됨 (별도 기록 필요 없음)
   - 구간별 한국 랭킹 (스트라바보다 상세함)
   - 탄소 감축량 자동 계산
   - 무료 (프리미엄은 월 5,900원)

   여러분을 라이트브라더스 "앰배서더"로 모시고 싶습니다.
   - 월 SSP 50,000 보너스
   - 앱에서 "공식 앰배서더" 뱃지 표시
   - 1년에 1회 번디 또는 자전거용품 선물

   관심 있으신 분은 회신 부탁드립니다!
   ```
3. 초기 20명 확보 목표

**기대 효과:**
- 스트라바 팔로워 50명 × 평균 팔로워 수 3,000명 = 15만명의 노출
- 전환율 1~2% = 1,500~3,000명 신규 가입

### 2.4 전술 3: 자전거 카페 및 온라인 커뮤니티 공략

**네이버 카페 전략:**
- 상위 자전거 카페 50개 선정 (멤버 1,000명 이상)
- 각 카페 "공지사항"에 라이트브라더스 소개 게시물 (카페장 협의)
  ```
  [공식 공지] 라이트브라더스 새 앱 - 자전거 기록 + 탄소 감축

  우리 카페 회원분들을 위해 특별 협약을 체결했습니다.
  - 정기 라이드 시 참가 기록 자동 인정
  - 카페별 대항전 자동 개최
  - 카페 추천인 링크로 가입 시 추가 SSP 1,000

  더 알아보기: [링크]
  ```
- 카페장 인센티브: SSP 100,000

**바이크포럼, 클리앙 자전거 게시판:**
- 정기 광고 게시 (주 2회, 유지비 월 20만원)
- 사용자 질문에 CEO 또는 직원이 직접 답변 (신뢰도 향상)

### 2.5 전술 4: 자전거 매장 오프라인 QR 코드 (위에서 언급)

**집행:**
- 전국 자전거 전문점 200개 선정
- 각 매장에 A4 포스터 설치 (무료, 라이트브라더스 로고 포함)
- 매달 업데이트: "이 달의 추천 세그먼트", "최고 기록자 소개"
- 매장장 인센티브: 월 50,000원 상당 자전거 부품 제공

#### 기대 효과 (섹션 2 전체)
- **동호회 파트너:** 100개 크루 × 평균 30명 = 3,000명
- **스트라바 경로:** 1,500~3,000명
- **카페/온라인:** 2,000~3,000명
- **소계:** 6,500~9,000명 (목표 10,000의 65~90%)

## 3. 러닝 크루 시장 공략 (Running Crew Market)

### 3.1 한국 러닝 문화 분석
- **한국 러닝 인구:** 약 1,000만명 (국민의 20%)
- **정기적 러닝 동호인:** 약 300만명 (월 4회 이상)
- **러닝 크루 (구조화된):** 약 2,000~3,000개 (5~100명)
- **카카오 러닝 크루 서비스 사용자:** 약 100만명
- **마라톤 참가자/년:** 약 100만명 (400개 대회)

### 3.2 전술 1: 서울 주요 러닝 크루 공략

**목표:** 서울 및 광역시 상위 러닝 크루 200개 확보

**실행:**
1. 러닝 크루 리스트업
   - 인스타그램 해시태그: #서울러닝크루 #마라톤동호회 #런데이 등으로 500개 크루 발굴
   - 카카오톡 비공개 그룹에서 활성 크루 확인
   - 상위 200개 선정 (멤버 10명 이상, 월간 활동 4회 이상)

2. 크루 리더 개인 연락 (CEO + 1명 마케팅)
   - 인스타그램 DM
   - 카톡 (크루 공개 번호가 있으면)
   - 이메일

3. 오프라인 미팅: "러닝크루 네트워킹 이벤트"
   - 장소: 올림픽공원 (서울 중심)
   - 일시: 매주 토요일 오전 6:00 (러닝 크루들이 자연스럽게 모이는 시간)
   - 형식:
     - 06:00~07:00: 함께 5km 런 (모든 크루 합동)
     - 07:00~08:00: 브런치 (크루 리더들과 네트워킹)
     - 08:00~09:00: "라이트브라더스 앱 소개" (15분) + Q&A
   - CEO 직접 참석 (매주)
   - 비용: 식사 + 음료 = 회당 50만원 (4주 × 4 = 월 200만원)

4. 파트너 크루 협약
   - 크루 이름 등록, 크루별 대항전 자동 개최
   - 월간 크루 랭킹 발표
   - 1등 크루: 상품 (자전거 라이트, 러닝 갓 등 5만원 상당)

```
라이트브라더스 CEO
```

## 6. 입소문 메커니즘 (Viral Mechanics)

### 6.1 추천 시스템: "친구 초대하면 SSP 2,000"

**메커니즘:**
```
A가 B를 초대 (고유 링크로)
→ B가 가입 & 첫 활동 기록
→ A에게 SSP 2,000 지급
→ B도 SSP 2,000 지급

최대 초대 횟수: 무제한 (하지만 1명당 1회만 카운트)
SSP 출처 태그: "REFERRAL" (나중에 추적 가능)
```

**배포 지점:**
1. 앱 "나의 페이지" → "친구 초대" 버튼
   - 링크 복사
   - 카톡, 인스타 DM 공유 버튼
   - 초대 현황 표시 ("초대한 사람: 5명, 가입한 사람: 3명, 받은 SSP: 6,000")

2. 앱 "홈" 화면 하단 배너
   - "친구를 초대하고 SSP 받으세요"
   - 매주 1회 (요일별로 다른 배너)

3. 이메일/카톡 자동 발송
   - 가입 후 3일: "친구들도 라이트브라더스에 초대해보세요!"
   - 이후 주 1회

**예상 효과:**
- 기존 회원 20%가 평균 2명 초대
- 13만 명 × 20% × 2명 = 52,000명 이차 추천
- 가입율 50% (링크 클릭했지만 설치 안 한 사람 고려) = 26,000명
- **목표 10,000 MAU 중 25%는 이 채널에서**

**SSP 비용:**
- 52,000 초대 × 2,000 SSP × 환산가 100원/SSP = 1,040만원 (1차 + 2차)

### 6.2 SNS 공유 카드: "오늘의 탄소저감량 자동 생성"

**기능:**
매일 자정에 자동으로 사용자별 SNS 공유 카드 생성:


**배포:**

**예상 효과:**

- 클릭율 2% = 신규 가입 유도

### 6.3 크루 초대 시스템

**기능:**
- 사용자가 크루를 생성할 때, 크루명 기반 고유 초대 코드 생성
- 크루원 초대 시 SSP 1,000 지급 (초대자)
- 크루원 가입 시 SSP 1,000 지급 (초대된자)

**예시:**
```
성수 자전거 크루 → 초대 코드: SSD-BIKE-2026
리더가 10명 초대 → 리더가 SSP 10,000 받음 + 각 멤버 SSP 1,000
→ 크루가 점점 커짐 → 크루 대항전 자동 개최 → 더 활동 활발 → 입소문
```
