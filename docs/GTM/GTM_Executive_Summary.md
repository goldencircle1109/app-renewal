## 5. 주요 위험 요소 & 완화 전략

| 위험 | 확률 | 영향 | 완화 전략 |

**결론:** 1순위 위험 2개(신규 가입, 앱 품질)에 집중 대응하면 목표 달성 가능

## 6. 성공의 황금률

```
성공 = 좋은 제품 × 명확한 메세징 × 올바른 채널 × CEO의 시간 투자

✓ 좋은 제품: Phase 1 앱 90% 안정성 (버그 <10개)
✓ 메세징: 모든 채널에서 "탄소+보상+커뮤니티" 일관성
✓ 올바른 채널: 기존 회원(25%) + 동호회(50%) + 기업(15%) + 광고(10%)
✓ CEO의 시간: 매주 동호회 리더 미팅 (주 20시간)
```

## Appendix: 자주 받는 질문 (FAQ)

**Q: 월 500만원 마케팅 예산으로는 불가능한가?**

A: 맞습니다. 13,800만원 필요합니다. 하지만 대안이 있습니다:
1. 펀딩 조달 (Pre-seed 3~5억원)
2. 무비용 마케팅 집중 (CEO 오프라인 + 자체 콘텐츠)
3. 예산 조정: 목표를 8,000 MAU로 낮추고 6개월로 연장

**Q: 기존 13만명의 몇 %를 전환하나?**

A: 약 2.5~3% (3,250명). 낮은 이유는:
- 앱 변경 피로도
- 기존 WB3 앱의 가치(포인트 이관)가 약함
- 새 기능을 이해하는 데 시간 필요

하지만 이 3,250명이 "씨앗"이 되어 추천으로 11,200명 더 유입

**Q: CEO가 이 일정을 다 할 수 있나?**

A: 어렵습니다. 권장:
- CEO: 전략 + 동호회 접촉 (주 20시간)
- 마케팅 담당: SNS + 광고 + 분석 (주 40시간)
- 추가 인턴/프리랜서: 콘텐츠 제작 (주 20시간)

최소 3명 팀 필요 (1명 정규 + 2명 파트타임/프리랜서)

**Q: 경쟁사(Strava, NRC)는 뭔가?**

A: 위협도 있지만, 우리의 강점이 더 많습니다:
- Strava: 글로벌 앱이지만 한국 로컬 기능 약함
- NRC: 니케 브랜드지만 크루 기능이 거의 없음
- 우리: 한국 문화 + 탄소 감축 + 크루 중심 (독점)

3개월이 초기 우위를 확보하는 데 충분

**Q: 프리미엄 구독 5% 전환율은 현실적인가?**

A: 보수적인 추정입니다. 근거:
- Strava: 프리미엄 전환율 10~15% (고가격 모델)
- 우리: 월 5,900원 (낮은 가격)
- 도전형 러너/사이클리스트: 기꺼이 지불 (기록 비교 기능)

실제로는 10~15% 가능성도 있음

**Q: 탄소 감축이 실제로 검증 가능한가?**

A: Phase 1에서는 부분적으로, Phase 4에서 완전히:
- Phase 1: T-map API로 자동차 기준 거리 계산 (추정값)
- Phase 2~3: 사용자 행동 패턴 학습
- Phase 4: MRV 3-Tier 인증 (자동/반자동/행동기반)

초기에는 "증명 가능성"이 아니라 "개인의 만족감" 중심

**Q: 지자체 파트너십이 실제로 가능한가?**

A: 가능합니다. 근거:
- 지자체: ESG 정책 달성 어려움 (정부 압박)
- 우리 앱: 자동으로 탄소 감축 측정 (지자체 입장에서 손쉬움)
- 비용 거의 없음 (우리가 기술 제공, 지자체는 인센티브만)

처음 3개 지자체는 어렵지만, 이후는 입소문으로 전파
