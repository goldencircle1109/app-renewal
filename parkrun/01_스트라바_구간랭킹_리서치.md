**작성일: 2026-03-19**

## 1. Strava 세그먼트란?

### 기본 개념

Strava 세그먼트(Segment)는 도로나 트레일의 **특정 구간을 미리 정의**해 두고, 해당 구간을 통과하는 모든 운동 기록을 **자동으로 감지하여 순위를 매기는 시스템**이다. 자전거, 달리기, 등산 등 다양한 종목에 적용된다.

### 세그먼트 생성 방식

| 방식 | 설명 |
|------|------|
| 사용자 직접 생성 | 기존 활동 기록에서 시작점/끝점을 지정하여 구간 생성. 공개/비공개 설정 가능 |
| Strava 자동 생성 | 많은 사용자가 반복 통과하는 구간을 Strava가 자동으로 세그먼트로 등록 |

### 세그먼트 매칭 작동 원리

1. 활동이 업로드되면 Strava 알고리즘이 **GPS 데이터를 분석**
2. 기존 세그먼트 데이터베이스와 **자동 대조**
3. 경로가 세그먼트의 시작점~끝점을 통과하면 **소요 시간 자동 계산**
4. 별도 사용자 조작 없이 **리더보드에 자동 기록**

## 2. 랭킹 시스템 상세

### KOM / QOM (King/Queen of the Mountain)

- 특정 세그먼트에서 **남성(KOM) / 여성(QOM) 최단 기록** 보유자에게 부여되는 칭호
- **왕관(crown) 아이콘**으로 시각적 표시
- 더 빠른 기록이 등록되는 순간 **실시간으로 칭호 이전** → 동적 순위

### CR (Course Record)

- 성별 구분 없이 해당 세그먼트의 **절대 최고 기록자**에게 부여

### 로컬 레전드 (Local Legend) — 2020년 도입

- **속도가 아닌 빈도를 보상**하는 시스템
- 최근 **90일 롤링 윈도우** 내 특정 세그먼트를 가장 많이 완주한 사람에게 월계관 아이콘 부여
- 빠르지 않아도 **꾸준히 같은 구간을 이용하는 사용자를 포용**하는 설계
- 커뮤니티 참여도를 높이는 핵심 역할

### 리더보드 필터링

| 필터 | 설명 | 무료/유료 |
|------|------|----------|
| 성별 필터 | 남/녀 구분 | 유료 |
| 연령대 필터 | 연령대별 순위 | 유료 |
| 팔로워만 보기 | 내 팔로워 내 순위 | 유료 |
| 클럽 내 순위 | 소속 클럽 내 순위 | 유료 |
| 기간별 필터 | 이번 주/월 등 | 유료 |
| 본인 기록 | 나의 기록 추이 | 무료 |

## 3. 기기 연동 방식

### Garmin 연동

- Garmin Connect 계정 연결 → 즐겨찾기(Star)한 세그먼트 **최대 100개** 기기에 자동 동기화
- 활동 기록은 Garmin Connect → Strava로 자동 업로드

### Apple Watch 연동

- 2024년 watchOS 10 이상 기기에서 **라이브 세그먼트 기능** 전면 지원
- 즐겨찾기 세그먼트 접근 시 시계 진동 알림 + 자동 라이브 세그먼트 화면 전환

### 라이브 세그먼트 (Live Segments)

- 세그먼트 구간 달리는 도중 **실시간으로 PR/KOM 대비 앞서는지** 표시
- 무료: 앞/뒤 여부만 확인 / 유료: 정확한 시간 차이 표시
- 달리기, 자전거 종목만 지원 (내리막 세그먼트 제외)

### 호환 기기

Garmin, Apple Watch, Wahoo, COROS, Hammerhead, ActiveLook 등

- 활동 기록/업로드, 세그먼트 자동 매칭, 기본 순위 확인

## 8. 기술 아키텍처

### GPS 매칭 알고리즘

1. 활동 업로드 → GPS 트랙을 세그먼트 DB와 대조
2. 세그먼트 시작점/끝점에 가장 가까운 GPS 포인트 선택
3. 경로 겹침이 충분하면 매칭 판정 → 소요 시간 계산

**설계 특성:**
- GPS 오차 수용을 위해 **느슨한 매칭 허용 범위** 적용
- 활동 방향이 세그먼트 생성 방향과 **일치해야** 매칭
- 종목 유형(달리기/자전거)이 **일치해야** 매칭

### 부정행위 감지 (머신러닝)

- 잘못된 종목 업로드 감지 (자동차 → 자전거 등)
- 비정상 GPS 데이터 (신호 불량, 드리프트) 감지
- e-bike/전동킥보드 사용 감지

**2024년 실적:** e-bike 의심 230만 건 + 차량 의심 160만 건 리더보드에서 제거 → 29만 3천 명 정당한 순위 회복

## 9. Strava Metro (B2B)

- 사용자 데이터를 **익명화/집계**하여 도시 계획자, 지방정부에 제공
- 세계 최대 **인간 동력 교통 데이터셋**
- 도시/트레일 계획 기관에는 **무료 제공**
- 자전거 도로, 보행자 인프라 투자 우선순위 결정에 활용

## 10. 한계 및 비판

### 안전 문제

- 2010년 캘리포니아: KOM 기록 도전 중 낙차 **사망 사고** 발생
- 영국: 4명 사망 사고 후 내리막 구간 리더보드 삭제
- **대응:** 위험 구간 KOM/QOM 미부여, 내리막 라이브 세그먼트 제외

### e-bike 부정행위

- 전기자전거 기록이 리더보드 공정성 훼손
- 자동 감지 시스템의 역오류(정상 기록 삭제)도 발생

### GPS 정확도 한계

- 보간 기술 한계로 밀리초 단위 정밀 측정 불가
- 기기 성능, 날씨, 샘플링 속도에 따라 기록 차이 발생

### 개인정보 보호

- 2018년 글로벌 히트맵에서 군사기지 주변 운동 경로 노출 사건
