# 한국 러닝/피트니스 시장 리서치

**작성일: 2026-03-19**
**작성 목적: 라이트브라더스 앱 리뉴얼 — 한국형 파크런 사업 타당성 분석**

---

## 1. 한국 러닝 시장 현황

### 러닝 인구 추이

| 연도 | 추정 러닝 인구 | 비고 |
|------|--------------|------|
| 2019 | 약 350만 명 | 코로나 이전 |
| 2024 | 약 550만 명 | 업계 추산 500~600만 |
| 2025 | **1,000만 명 돌파** | 추산 |

- 조깅·달리기 연간 경험률: 2021년 23% → 2023년 **32%** (타 종목 대비 최대 폭 증가)
- '러닝' SNS 검색량: 전년 대비 **270% 급증**
- '#런스타그램' 해시태그: 약 **400만 개 이상**

### 성장 배경

- 코로나19 이후 MZ세대 중심 폭발적 성장
- **'헬시플레져(Healthy Pleasure)'** 트렌드와 결합
- 러닝 = 단순 운동 → **라이프스타일 + 소셜 활동**으로 진화
- 진입 장벽 낮음 (러닝화+옷만 있으면 됨)
- SNS 공유 문화와 맞물려 2030세대 강력 지지

### 경제적 파급 효과

| 항목 | 규모 |
|------|------|
| 운동화 시장 | 2024년 **4조 원 돌파** (2021년 2.8조) |
| 러닝화 시장 | 2025년 **1조 원 돌파** |
| 러닝화 검색량 증가 | 전년 대비 +53% |
| 러닝 관련 소비 증가 | 30대 +232%, 40대 +225% |
| 춘천마라톤 지역경제 효과 | 요식업 +25.5%, 숙박업 +61.8% |

## 2. 한국 주요 러닝 앱

### 글로벌 앱

| 앱 | 특징 | 한국 내 위상 |
|----|------|------------|
| 나이키 런 클럽 (NRC) | GPS 기록, 가이드 런, SNS 공유 | 가장 대중적, "러닝계의 인스타그램" |
| 스트라바 (Strava) | SNS형 기록, 세그먼트 경쟁 | 40대·남성·고급 러너 비중 높음 |
| 가민 커넥트 | 가민 기기 연동, 상세 데이터 | 스트라바와 중복 설치율 25% |
| 아식스 런키퍼 | 맞춤 훈련, 다기기 연동 | 국내 인지도 낮음 |
| 삼성 헬스 | AI 코치, 심박수·산소포화도 | 삼성 기기 기본 탑재 |

### 국내 앱

| 앱 | 특징 |
|----|------|
| **런데이** | 러닝 입문자 특화, '30분 달리기 도전', MZ 여성 인기 |
| **러너블** | 오프라인 대회 정보 + 티켓 예매 + 완주 기록 인증 통합 |
| **런톡** | 러닝 전문 미디어 커뮤니티, 대회·훈련·기어 리뷰 |

### 시장 공백 (핵심 기회)

> **크루 관리 + 대회 등록 + 러닝 기록 + 커뮤니티를 원스톱으로 제공하는 서비스가 없다**
> 파크런 형태의 **정기적·무료 러닝 이벤트 플랫폼**도 미개발 상태

## 4. 한국 러닝 크루 문화

### 규모

- 서울 내 활동 러닝크루: **100개 이상** (2022년 기준, 이후 더 증가)
- 네이버 밴드 러닝·걷기 모임: 최근 2년 새 **46% 증가**
- 당근마켓 러닝 모임: 2024년 1~8월 **4배 이상** 증가

### 주요 러닝 크루 (서울)

| 크루명 | 특징 |
|--------|------|
| 88Seoul (EES) | 2015년 창설, 서울 길거리 달리기 최초, 패션·음악·아트 크로스오버 |
| JSRC (잠실러닝클럽) | 2014년 창설, 지역 건강 문화 구축 |
| 서울비너스 | 매주 금요일 저녁, 성별·경험 무관 |
| 7979 서울러닝크루 | 서울시 공식, 329명→898명 (2.7배 성장) |

### 주요 러닝 크루 (부산)

| 크루명 | 특징 |
|--------|------|
| ONNA RUN | '온전히 나를 위한 러닝' 웰니스 |
| PTR | 수·토·일 정기 훈련, 트레일+마라톤 통합 |

### ⚠️ 규제 이슈

- 서울 서초구: 반포종합운동장 **5인 이상 단체 달리기 제한** (2024.10~)
- 주거지 인근 소음·혼잡 민원 증가 → 일부 지자체 규제 강화 추세
- **사업 기획 시 장소 선정과 참가 인원 관리 주의 필요**

- 파크런 **한국 미진출** (글로벌 23개국 운영, 한국 미포함)

## 9. 스포츠 스폰서십 시장

### 브랜드 동향

| 브랜드 | 동향 |
|--------|------|
| 아디다스 | 한국을 **독립 단독 마켓으로 격상** (2024.1), 한국 특화 마케팅 본격화 |
| 나이키 | DTC 전략 조정 중, 러닝 이벤트 스폰서 유지 |

| 푸마 | 런푸마팸(RUN PUMA FAM) 러닝크루 마케팅 |

### 스폰서십 트렌드

1. 러닝 대회 주관/공동 후원 확대
2. 러닝 크루 협찬 (제품 + 공동 이벤트)
3. 여성 러너 타겟 마케팅 강화
4. 식음료 브랜드 약진

## 10. 규제 환경

### 주요 법규

| 법령 | 내용 |
|------|------|
| 도시공원 및 녹지 등에 관한 법률 | 공원 내 행사·시설 설치 기본법 |
| 서울시 도시공원 조례 | 서울 공원 행사 세부 기준 |
| 체육시설의 설치·이용에 관한 법률 | 체육 이벤트 시설 규정 |

### 한강공원 행사 신청

- 서울시 미래한강본부를 통해 **일반행사/체육행사 장소사용 신청**
- 지정 장소에서 **사전 허가** 후 행사 개최 가능
- 공공-민간 협력 형태도 가능

### ⚠️ 주의사항

- 대규모 행사: 서울시 허가 필수
- 도로 러닝: 경찰청 **도로 사용 허가** 별도 필요
- 주거지 인근: 소음·혼잡 민원 → 규제 강화 추세
