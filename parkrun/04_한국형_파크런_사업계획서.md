# 라이트브라더스 한국형 파크런 사업계획서

**"WB RUN" — 스트라바의 디지털 경쟁 + 파크런의 커뮤니티를 결합한 한국형 러닝 플랫폼**

**작성일: 2026-03-19**
**작성: 라이트브라더스 앱 리뉴얼 프로젝트**

---

## Executive Summary

### 기회

- 한국 러닝 인구 **1,000만 명** 돌파, 러닝화 시장 **1조 원**
- 호주 파크런 모델(1,100만 명 글로벌 등록)이 전 세계로 확산 중이나 **한국 미진출**
- 한국에 체계화된 **정기 무료 러닝 이벤트 플랫폼**이 없음
- 기존 앱(NRC, 스트라바)은 **한국 로컬 기능이 약함**

### 솔루션

라이트브라더스 앱 리뉴얼에 **"WB RUN"** 기능을 탑재하여:

1. **파크런의 커뮤니티 모델**을 한국 러닝 크루 문화에 맞게 현지화
2. **스트라바의 구간 랭킹(세그먼트)**을 한국 주요 러닝 코스에 적용
3. 파크런이 놓친 **디지털화 공백**을 완전한 디지털 경험으로 채움

### 목표

| 항목 | 1년차 | 3년차 | 5년차 |
|------|------|------|------|
| 등록 사용자 | 5만 명 | 30만 명 | 100만 명 |
| 주간 정기 런 개최지 | 수도권 10곳 | 전국 50곳 | 전국 150곳 |
| 월간 활성 사용자(MAU) | 1.5만 명 | 10만 명 | 40만 명 |

### 1.1 한국 러닝 시장

| 지표 | 수치 | 출처 |
|------|------|------|
| 러닝 인구 | 1,000만 명 (2025) | 업계 추산 |

| 디지털 피트니스 앱 CAGR | 19.17% (2025~2035) | Spherical Insights |

### 1.2 타겟 시장 규모 (TAM → SAM → SOM)

| 구분 | 규모 | 산정 근거 |
|------|------|---------|
| **TAM** (전체 시장) | 1,000만 명 | 한국 전체 러닝 인구 |
| **SAM** (접근 가능 시장) | 200만 명 | 수도권+광역시 러닝앱 사용 러너 |
| **SOM** (확보 목표 시장) | 5~30만 명 | 1~3년차 현실적 확보 목표 |

### 1.3 경쟁 환경

| 경쟁사 | 강점 | 약점 (= 우리의 기회) |
|--------|------|---------------------|
| NRC | 브랜드, 대중성 | 로컬 크루 기능 없음, 커뮤니티 약함 |
| 스트라바 | 세그먼트, 고급 데이터 | 유료 많음, 대중 접근성 낮음, 이벤트 기능 없음 |
| 가민 커넥트 | 데이터 정밀성 | 가민 기기 필수 |
| 런데이 | 초보자 친화 | 커뮤니티/대회 기능 제한 |
| 러너블 | 대회 정보 통합 | 크루/GPS 기능 없음 |

### 1.4 시장 공백 (핵심)

> **크루 관리 + 정기 런 이벤트 + GPS 기록 + 구간 랭킹 + 커뮤니티를 하나로 통합한 서비스가 없다**

## 3. 핵심 기능 상세

### 3.1 WB 파크런 (커뮤니티 런)

파크런의 핵심 모델을 한국에 맞게 디지털 업그레이드

| 파크런 (원본) | WB RUN (디지털 업그레이드) |
|-------------|------------------------|
| 물리적 바코드 등록 | **QR코드 + NFC** 디지털 체크인 |
| 플라스틱 완주 토큰 | **자동 GPS 완주 감지** + 디지털 배지 |
| 수 시간 후 결과 | **실시간 리더보드** (완주 즉시) |
| 이메일로 봉사자 모집 | **앱 내 봉사자 매칭 시스템** |
| 토큰 수동 정렬 | **완전 자동 순위 처리** |
| 사후 기록 조회만 가능 | **라이브 트래킹** (친구/가족이 실시간 위치 확인) |

**운영 방식:**
- **매주 토요일 오전 8시** (한국 시간)
- **5km 코스** (한강, 올림픽공원, 각 도시 공원 등)
- **참가비 무료** (파크런 핵심 원칙 유지)
- 사전 앱 체크인 → GPS 자동 트래킹 → 실시간 순위 → 완주 즉시 결과

### 3.2 구간 랭킹 (스트라바 영감)

스트라바의 세그먼트 랭킹을 한국 러닝 코스에 최적화

**사전 등록 구간 (Phase 1 — 서울 10개 코스):**

| 구간명 | 거리 | 위치 |
|--------|------|------|
| 여의도 한강 순환 | 6.8km | 여의도 한강공원 |
| 반포 한강 야경 | 6.8km | 반포 한강공원 |
| 잠수교-한강대교 | 10km | 잠수교~한강대교 |
| 올림픽공원 호반의길 | 5km | 올림픽공원 |
| 남산 둘레 | 7km | 남산 |
| 서울숲 순환 | 4.5km | 서울숲 |
| 상암 월드컵공원 | 5km | 월드컵공원 |
| 뚝섬 한강 | 5km | 뚝섬 한강공원 |
| 탄천 10K | 10km | 탄천 |
| 북한산 둘레길 입문 | 3km | 북한산 |

**랭킹 시스템:**

| 기능 | 설명 |
|------|------|
| **코스 킹/퀸** | 각 구간 남/녀 최고 기록 → 왕관 아이콘 (스트라바 KOM/QOM) |
| **단골 러너** | 90일 내 최다 완주 → 월계관 아이콘 (스트라바 로컬 레전드) |
| **연령대별 순위** | 20대/30대/40대/50대+ 필터 |
| **크루별 순위** | 소속 크루 내 순위 |
| **시즌 랭킹** | 분기별 종합 순위 + 시즌 리워드 |
| **라이브 세그먼트** | 달리는 도중 실시간 PR/코스킹 대비 표시 |

### 3.3 크루 허브 (소셜 기능)

한국 러닝 크루 문화를 디지털로 통합

| 기능 | 설명 |
|------|------|
| 크루 생성/관리 | 러닝 크루를 앱에서 직접 생성, 멤버 관리 |
| 크루 정기 런 | 크루 자체 정기 런 일정 관리 + 자동 기록 |
| 크루 대항전 | 크루 vs 크루 주간/월간 챌린지 |
| 크루 통합 리더보드 | 크루 전체 누적 거리, 평균 페이스 등 |
| 소셜 피드 | 러닝 기록 공유, 쿠도스(좋아요), 댓글 |
| 크루 검색/가입 | 지역/실력/취향별 크루 검색 및 가입 |

### 4.2 무료 vs 프리미엄

**무료 (WB RUN Basic)**
- WB 파크런 참가 (주간 정기 5K)
- 기본 GPS 기록 및 구간 순위
- 크루 가입 (최대 1개)
- 소셜 피드, 쿠도스
- 기본 개인 기록 조회

**프리미엄 (WB RUN Pro) — 월 5,900원 / 연 49,900원**
- 구간 랭킹 상세 필터 (연령대, 크루, 기간별)
- 라이브 세그먼트 (실시간 PR/코스킹 대비)
- 고급 훈련 분석 (페이스존, 심박존, GAP)
- 크루 무제한 가입 + 크루 생성
- 크루 대항전 개최
- 시즌 랭킹 참여 + 리워드
- 가민/애플워치 고급 연동
- 광고 제거

## 13. 차별화 전략 요약

### "왜 WB RUN인가?"

| vs 경쟁사 | WB RUN 차별점 |
|----------|-------------|
| vs 파크런 (원본) | **완전한 디지털화** — 실시간 결과, 라이브 세그먼트, GPS 통합 |
| vs 스트라바 | **오프라인 이벤트** — 매주 실제 만나서 달리는 커뮤니티 |
| vs NRC | **한국 로컬 최적화** — 크루 관리, 한국 코스 구간 랭킹 |
| vs 런데이 | **경쟁+커뮤니티** — 랭킹 시스템 + 크루 대항전 |
| vs 러너블 | **통합 플랫폼** — 이벤트+기록+크루+랭킹+커머스 원스톱 |

> **"스트라바의 디지털 경쟁력과 파크런의 커뮤니티 매력을 결합하여,
> 한국 1,000만 러너를 위한 최초의 통합 러닝 플랫폼을 만든다"**
