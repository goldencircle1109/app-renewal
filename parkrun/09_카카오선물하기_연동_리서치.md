## 2. 가입 절차 (Step by Step)

1. **giftbiz.kakao.com** 접속 → "시작하기"
2. 카카오 계정 로그인 → 카카오 비즈니스 약관 동의
3. **사업자등록번호** 입력 및 인증
4. 사업자등록증 사본 첨부, 대표자 정보 입력
5. **비즈 그룹** 생성 (그룹명 지정, 마스터 권한 획득)
6. 환급 계좌 인증
7. **비즈캐시 충전** (1회 1,000원 ~ 10억원 미만, 법인카드 가능)
8. 서비스 이용 시작

**공식 이용안내 PDF**: https://st.kakaocdn.net/commerce_ui/static/front-biz-gift/선물하기%20for%20Biz_이용안내_v1.4.pdf

## 5. API 연동 상세

### 5.1 API 키 발급

1. giftbiz.kakao.com 로그인
2. **[마이비즈] → [그룹 정보 관리]** 클릭
3. "API 인증키 발급" (마스터 권한 필수)

### 5.2 주요 API

| API | 설명 |
|-----|------|
| 템플릿 조회 | 발송 가능한 모바일 교환권 상품 목록 |
| 상품 조회 | 특정 상품 정보 조회 |
| **주문/발송** | 수신자 전화번호 기반 선물 발송 |
| 발송 현황 조회 | 수신/사용/미사용/취소 상태 확인 |
| 주문 취소/환불 | 미사용 주문 취소 처리 |

### 5.3 발송 플로우

```
라브 앱: 사용자 SSP 교환 요청
    │
    ▼
라브 서버: SSP 차감 + 기프트비즈 API 호출
    ├── Header: API 인증키
    ├── Body: 수신자 전화번호, 상품 템플릿 ID, 메시지
    │
    ▼
카카오 서버: 카카오톡으로 선물 발송
    ├── 카카오 가입자 → 카카오톡 메시지
    └── 미가입자 → MMS 대체 발송
    │
    ▼
콜백 수신: 발송/수신/사용 상태 추적
```

### 5.4 수신자 정보

- **휴대폰 번호** 기반 발송 (카카오 ID 불필요)
- 카카오톡 미가입자에게도 **MMS 대체 발송** 가능 (모바일 교환권 한정)

### 5.5 콜백/웹훅

- 상태 변경 시 등록된 URL로 HTTP POST
- **3초 이내 200 OK** 응답 필요

### 5.6 테스트 환경

- 별도 샌드박스 **없음** — 실서비스에서 소액 상품으로 테스트

## 6. 발송 방식

| 방식 | 최대 건수 | 용도 |
|------|----------|------|
| **API 자동 발송** | 무제한 (rate limit 내) | 실시간 포인트 교환 |
| 엑셀 업로드 | 1회 **5,000명** | 이벤트 일괄 발송 |
| 예약 발송 | 날짜/시간 지정 | 정기 캠페인 |

## 7. 비용 구조

| 항목 | 내용 |
|------|------|
| 중개 수수료 | **없음** (상품 정가 그대로) |

## 8. 미사용 환불 정책

주문 시 2가지 옵션 선택:

| 옵션 | 만료 시 처리 |
|------|------------|
| **옵션 A (권장)** | 기업에게 비즈캐시 **100% 환급** |
| 옵션 B | 수신자에게 카카오 쇼핑포인트 100% 지급 |

- 선물 거절/배송지 미입력 → 비즈캐시 **100% 자동 환불**

## 9. 카카오페이 포인트 직접 지급 가능성

### 9.1 머니 전송 링크 API

- 카카오페이 개발자센터: developers.kakaopay.com
- 기업이 **카카오페이 머니 전송 링크**를 생성 → 수신자가 링크 클릭 후 수령
- **파트너 계약 필요**: biz.kakaopay.com
- 필요 서류: 사업자등록증, 정산계좌, 대표자 신분증

### 9.2 카카오페이 포인트 직접 지급

| 방법 | 가능 여부 | 조건 |
|------|----------|------|
| 선물하기 for Biz (교환권) | **가능** | API 즉시 사용 |
| 머니 전송 링크 API | 가능 | 카카오페이 파트너 계약 |
| 페이포인트 직접 지급 | 공개 API 없음 | 별도 B2B 계약 |
