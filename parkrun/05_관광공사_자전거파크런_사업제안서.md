# 한국관광공사 × 라이트브라더스 사업제안서

## "대한민국 자전거 파크런" — 디지털 스탬프 투어 × 구간 랭킹 × ESG 캠페인

**제안사: (주)라이트브라더스**
**수신: 한국관광공사 지역콘텐츠육성팀 심경섭님**
**작성일: 2026년 3월 19일**

---

## 1. 제안 요약 (Executive Summary)

한국관광공사 님께서 요청하신 3가지 협업 영역에 대해, 라이트브라더스의 **스윗스웻포인트(SSP)** 시스템과 호주 **파크런(parkrun)** 모델을 결합한 통합 솔루션을 제안드립니다.

| 관광공사 요청 사항 | 라이트브라더스 제안 솔루션 |
|-------------------|------------------------|
| 1. 대표코스 60선 + 순환코스 연계 스탬프 투어 | **디지털 스탬프 투어** — GPS 자동 인증 + 구간 랭킹 시스템 |
| 2. 전국 순환코스 완주 캠페인 | **자전거 파크런** — 매주 정기 라이딩 이벤트 + 완주 챌린지 |
| 3. ESG 협업 친환경 캠페인 | **스윗스웻포인트(SSP)** — 탄소저감 마일리지 보상 체계 |

### 핵심 가치 제안

> **"달릴수록 포인트가 쌓이고, 지역 관광이 활성화되고, 탄소가 줄어드는 선순환 플랫폼"**

## 2. 라이트브라더스 소개 및 협업 역량

### 2.1 회사 소개

라이트브라더스는 대한민국 대표 자전거 전문 플랫폼으로, **온라인 커머스 + 오프라인 커뮤니티 + 디지털 리워드** 시스템을 통합 운영하고 있습니다.

### 2.2 스윗스웻포인트(Sweet Sweat Point, SSP) — 핵심 기술 자산

| 항목 | 내용 |
|------|------|
| **개념** | 자전거 라이딩 기반 R2E(Ride to Earn) 보상 체계 |
| **작동 방식** | 스트라바(Strava) 연동 → 라이딩 기록 자동 감지 → 포인트 자동 적립 |
| **적립 기준** | 1km당 10포인트 (탄소 저감량 기반 계산) |
| **사용처** | 라이트브라더스 쇼룸/온라인몰에서 자전거 용품 구매 |
| **탄소 연계** | 자동차/대중교통 대신 자전거 탑승 시 줄어드는 탄소량을 포인트로 환산 |
| **기존 협력** | 춘천지속가능발전협의회 등 지자체 협업 실적 보유 |

### 2.3 공공기관 협업 실적

- **2024년 범정부 자전거 이용 활성화 MOU 참여**: 탄소중립녹색성장위원회 + 국토부 + 행안부 + 환경부 + 서울시 + 카카오모빌리티 + 티맵모빌리티 + **라이트브라더스** 공동 협약
- 춘천지속가능발전협의회 시민 자전거 이용 활성화 MOU
- ESG 기반 탄소저감 캠페인 다수 운영
- 공공자전거 정보 통합 제공 플랫폼 구축 참여

> **라이트브라더스는 이미 정부의 자전거 활성화 정책의 공식 파트너입니다.**
> 이번 관광공사 협업은 기존 정부 협력의 자연스러운 확장입니다.

## 3. 제안 사업 1: 디지털 스탬프 투어

### "대표코스 60선 + 순환코스 연계 스마트 스탬프 투어"

### 3.1 현재 스탬프 투어의 한계

| 기존 방식 | 문제점 |
|----------|--------|
| 물리적 스탬프 / 도장 찍기 | 인증센터 운영 비용, 위치 제약, 관리 인력 필요 |
| 종이 스탬프북 | 분실 위험, 데이터 축적 불가, MZ세대 참여 저조 |
| 별도 앱 없이 운영 | 참가자 추적 불가, 마케팅 데이터 부재 |

### 3.2 기존 국토종주 인증제와의 시너지

현재 행안부·환경부가 운영하는 **국토종주인증제**에는 이미 3중 인증 체계가 있습니다.

| 기존 인증 체계 | 현황 | 한계 |
|-------------|------|------|
| 오프라인 스탬프 | 유인 15개소 + 무인 70개소 | 종이 수첩 기반, 데이터 축적 불가 |
| QR 사이버인증 | 자전거행복나눔 앱 | 국토종주길 한정, 60선 미연계 |
| GPS 자동인증 | 인증센터 부근 자동 처리 | 구간 랭킹/경쟁 기능 없음 |

**라이트브라더스의 역할: 기존 인프라를 "보완"하여 60선까지 확장하고 게이미피케이션을 추가합니다.**

→ 기존 국토종주 인증 + 신규 60선 디지털 스탬프 + 구간 랭킹 + SSP 포인트를 **하나의 앱에서 통합 제공**

### 3.3 라이트브라더스 디지털 스탬프 투어 솔루션

```
┌──────────────────────────────────────────────────────────┐
│               디지털 스탬프 투어 시스템                      │
├──────────────┬──────────────────┬────────────────────────┤
│  GPS 자동 인증  │  구간 랭킹 시스템  │  SSP 포인트 적립       │
├──────────────┼──────────────────┼────────────────────────┤
│ 코스 진입 자동감지│ 코스별 시간 순위  │ 1km당 10포인트 적립     │
│ 체크포인트 통과  │ 연령대/지역별 랭킹 │ 탄소저감량 자동 계산    │
│ 완주 자동 인증  │ 시즌별 랭킹 집계  │ 지역 관광지 쿠폰 교환    │
│ 디지털 배지 발급 │ KOM/QOM 시스템   │ 완주 인증서 발급        │
└──────────────┴──────────────────┴────────────────────────┘
```

### 3.3 대표코스 60선 디지털 전환

**관광공사 대표코스 60선**을 앱 내 **공식 세그먼트(구간)**으로 등록합니다.

| 코스 카테고리 | 코스 수 | 디지털 전환 내용 |
|-------------|--------|----------------|
| 도심 | 다수 | GPS 구간 등록 + 자동 인증 + 도심 관광지 연계 |
| 바다 | 다수 | 해안 코스 + 주변 해산물 맛집 쿠폰 연계 |
| 섬 | 다수 | 섬 완주 특별 배지 + 숙박 할인 |
| 강 | 다수 | 강변 코스 + 자전거 카페 제휴 |
| 산 | 다수 | 난이도별 랭킹 + 등급 시스템 |
| 도전 | 다수 | 장거리 챌린지 + 특별 리워드 |
| 역사 | 다수 | 역사 해설 오디오 가이드 연계 |
| 힐링 | 다수 | 웰니스 프로그램 + 명상 스팟 안내 |

**기술적 구현:**
- 관광공사가 이미 제공하는 **GPX 파일**을 활용하여 세그먼트 자동 등록
- 스트라바 API 연동으로 기존 자전거 이용자 **즉시 참여 가능**
- 가민/애플워치 등 웨어러블 기기 연동

### 3.4 순환코스 디지털 스탬프

| 순환코스 권역 | 디지털 스탬프 설계 |

## 4. 제안 사업 2: 자전거 파크런 — 전국 순환코스 완주 캠페인

### "매주 같은 시간, 같은 장소 — 대한민국 자전거 파크런"

### 4.1 파크런(parkrun) 모델이란?

| 항목 | 내용 |
|------|------|
| 기원 | 2004년 영국에서 시작, 현재 23개국 1,100만 명 참가 |
| 방식 | 매주 토요일 오전, 같은 장소에서 5km 달리기 (무료, 자원봉사 운영) |
| 성공 요인 | 무료 + 정기적 + 누구나 참여 + 커뮤니티 |
| 호주 규모 | 100만 명 등록, 490개 장소, 연 375만 회 완주 |
| WHO 평가 | "지역 커뮤니티를 정기적 신체 활동에 참여시키는 **모범 사례**" |

### 4.2 자전거 파크런: 한국형 적용

파크런의 검증된 커뮤니티 모델을 **자전거 + 관광**에 맞게 변형합니다.

| 파크런 (러닝) | 자전거 파크런 (제안) |
|-------------|-------------------|
| 매주 토요일 5km 달리기 | **매주 토요일 오전, 대표코스 60선에서 자전거 라이딩** |
| 물리적 바코드 인증 | **GPS 자동 인증 + 디지털 체크인** |
| 종이 결과표 | **실시간 리더보드 + 구간 랭킹** |
| 무료 참가 | **무료 참가 + SSP 포인트 적립** |
| 자원봉사 운영 | **지역 자전거 동호회 + 관광공사 협력 운영** |

### 4.3 자전거 파크런 운영 모델

**Phase 1: 파일럿 (5개 거점)**

| 거점 | 코스 | 거리 | 특징 |
|------|------|------|------|
| 서울 한강 | 여의도~반포 순환 | 약 15km | 수도권 핵심, 접근성 최고 |
| 춘천 | 의암호 순환 | 약 20km | 기존 라이트브라더스 협력 지역 |
| 경주 | 역사유적길 | 약 26km | 역사 관광 + 자전거 최적 |
| 제주 | 해안도로 구간 | 약 20km | 관광 수요 최고 |
| 부산 | 낙동강 하구 | 약 15km | 남부권 거점 |

**운영 방식:**

```
매주 토요일 오전 9시
├── 08:30  현장 집결 + 앱 체크인
├── 08:50  안전 브리핑 + 코스 안내
├── 09:00  일제 출발
├── 라이딩  GPS 자동 트래킹 + 실시간 랭킹
├── 완주    자동 완주 인증 + SSP 포인트 적립
├── 완주 후  주변 관광지/맛집 쿠폰 자동 발급
└── 결과    실시간 리더보드 + 주간 랭킹 업데이트
```

### 4.5 구간 랭킹 시스템 (스트라바 영감)

각 대표코스에 **구간 랭킹**을 적용하여 경쟁과 재방문을 유도합니다.

| 기능 | 설명 |
|------|------|
| **코스 킹/퀸 (KOR/QOR)** | 각 코스 남/녀 최단 시간 기록 보유자 → 왕관 아이콘 |
| **단골 라이더** | 90일 내 특정 코스 최다 완주자 → 월계관 아이콘 |
| **연령대별 순위** | 20대/30대/40대/50대+ 필터 |
| **지역별 순위** | 시/도별 라이더 순위 → 지역 자존심 대결 |
| **시즌 랭킹** | 분기별 종합 순위 → 시즌 챔피언 시상 |
| **동호회 대항전** | 자전거 동호회 간 팀 랭킹 → 단체 참여 유도 |

## 5. 제안 사업 3: ESG 협업 친환경 캠페인

### "라이딩으로 지구를 지키다" — 탄소저감 캠페인

### 5.1 SSP 탄소저감 메커니즘

```
자전거 라이딩 1km
    ↓
탄소저감량 자동 계산 (자동차 대비 CO₂ 절감량)
    ↓
SSP 10포인트 자동 적립 (1km당)
    ↓
포인트로 자전거 용품 구매 또는 지역 관광 쿠폰 교환
    ↓
"달릴수록 이득, 달릴수록 친환경"
```

### 5.2 관광공사 × 라이트브라더스 ESG 캠페인 제안

**캠페인 1: "Green Ride Korea" — 국가 탄소저감 라이딩 캠페인**

| 항목 | 내용 |
|------|------|
| 목표 | 연간 자전거 관광 라이딩을 통한 국가 단위 탄소저감량 시각화 |
| 방식 | 대표코스 60선 라이딩 시 탄소저감량 자동 집계 → 전국 대시보드 공개 |
| KPI | 연간 탄소저감량 목표 설정 (예: 100톤 CO₂ 저감) |
| 시각화 | 관광공사 홈페이지 + 라이트브라더스 앱에 실시간 탄소저감 카운터 |
| 보상 | 목표 달성 시 참가자 전원 특별 배지 + SSP 보너스 포인트 |

**캠페인 2: "Zero Carbon Tour" — 탄소 제로 여행 인증**

| 항목 | 내용 |
|------|------|
| 목표 | 자전거로만 여행하는 "탄소 제로 관광" 인증 제도 |
| 방식 | 대표코스 + 주변 관광지를 자전거로만 이동 시 "Zero Carbon Tour" 배지 발급 |
| 연계 | 관광공사 품질인증 업소(숙박, 식당) 방문 시 추가 포인트 |
| ESG 보고 | 관광공사 ESG 보고서에 캠페인 실적 반영 가능 |

**캠페인 3: "Ride for Local" — 자전거 관광으로 지역경제 활성화**

| 항목 | 내용 |
|------|------|
| 목표 | 자전거 관광이 지역 경제에 기여하는 효과 측정 및 확대 |
| 방식 | 코스 완주 후 주변 지역 상점/식당 SSP 쿠폰 자동 발급 |
| 데이터 | 라이딩 데이터 기반 지역별 관광 기여도 분석 리포트 제공 |
| 참고 | 춘천마라톤 사례: 러닝 대회로 요식업 +25.5%, 숙박업 +61.8% 증가 |
