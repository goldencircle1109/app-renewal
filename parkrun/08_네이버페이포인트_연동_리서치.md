## 2. 다우기술 애드콘 — 1순위 권장

### 2.1 서비스 개요

2024년 6월 출시. 자체 포인트 제도를 운영하는 사업자가 네이버페이 포인트를 고객에게 실시간 지급할 수 있는 B2B 중계 서비스.

### 2.2 핵심 기능

- **네이버 ID 직접 지급**: 쿠폰 번호 입력 없이 네이버 ID로 즉시 적립
- **전환 비율 자유 설정**: SSP 100P = 네이버페이 50원 등 사업자 결정
- **지급 시나리오**: 미션 리워드, 이벤트 경품, CS 보상, 임직원 복지

### 2.3 연동 아키텍처

```
사용자 SSP 교환 요청
    │
    ▼
라이트브라더스 앱 서버 (REST API 호출)
    │
    ▼
다우기술 애드콘 API
    │
    ▼
네이버페이 포인트 계정에 즉시 적립
    │
    ▼
사용자 네이버페이 앱 알림 수신
```

### 2.4 기술 사양

- **방식**: REST API (HTTP, 서버 to 서버)
- **별도 SDK**: 불필요
- **인증**: API Key 기반
- **수신자 식별**: 네이버 로그인 ID (네이버페이 별도 가입 불필요)

### 2.5 수수료

- 공개 고정 요금제 없음 (계약 협상 방식)
- 예시 적용 요율: 결제금액의 **2.75%**
- 구조: 포인트 원가(1P=1원) + 서비스 이용료

### 2.6 파트너 등록 절차 (추정)

## 3. 기프티쇼 비즈 / 비즈콘 — 쿠폰 형태 2순위

### 3.1 네이버페이 포인트 쿠폰 상품

| 상품 | 용도 |
|------|------|
| 네이버페이 포인트쿠폰 3,000원 | 소액 리워드 |
| 네이버페이 포인트쿠폰 5,000원 | 일반 미션 보상 |
| 네이버페이 포인트쿠폰 10,000원 | 주요 이벤트 경품 |
| 네이버페이 포인트쿠폰 30,000원 | 대형 이벤트 경품 |

### 3.2 할인율

- 기프티쇼 비즈: 최대 **5%**
- 비즈콘: 최대 **12%** (볼륨 기준)

### 3.3 수령자 경험

1. 기업 → 사용자 핸드폰으로 쿠폰 번호 전달 (MMS/카카오톡)
2. 사용자가 네이버페이 앱 → 쿠폰 등록
3. 포인트 즉시 적립

**단점**: 쿠폰 번호 수동 등록 필요 (애드콘 대비 UX 열위)

### 3.4 쿠폰 유효기간

- 쿠폰 등록 마감: 구매일로부터 30~90일 (상품별 상이)
- 포인트 전환 후: **10년** 유효

## 4. 포트원(아임포트) 경유 — 리워드 부적합

### 4.1 API 엔드포인트

```
POST https://api.iamport.kr/payments/{imp_uid}/naver/point
```

### 4.2 적용 대상

네이버페이 **결제형 가맹점**에서 결제 완료 후 포인트를 소급 적립하는 기능. "리워드 지급"이 아닌 "결제 후 적립" 목적이므로 **라이트브라더스의 미션 보상 시나리오에는 부적합**.

## 5. 네이버페이 직접 제휴

### 5.1 개발자센터

- URL: https://developer.pay.naver.com/
- 가맹 유형: 주문형(스마트스토어), **결제형(독립몰/앱)**, 종합형

### 5.2 결제형 가맹

- 스마트스토어 없이 **앱만으로 결제형 가맹점 가입 가능**
- 기술 검수 15 영업일 소요
- 단, 이것은 "결제 수단 추가"이지 "포인트 리워드 지급"이 아님

### 5.3 포인트 지급형 직접 제휴

- 대기업/대형 플랫폼 중심 직접 계약 구조
- 스타트업은 **다우기술 애드콘 같은 대행사 경유가 현실적**

## 7. 법적/규제 고려사항

### 7.1 전자금융업 등록

애드콘/기프티쇼를 통해 **기존 발행된 포인트를 구매하여 지급**하는 경우 → 라이트브라더스는 발행자가 아니므로 **등록 불필요**

### 7.2 세금 처리 (기타소득 원천징수)

| 1회 지급액 | 처리 |
|-----------|------|
| **5만원 이하** | 비과세, 원천징수 없음 |
| **5만원 초과** | 기타소득 22% (기타소득세 20% + 지방소득세 2%) |

- SSP 일반 교환(1,000~5,000원): **대부분 비과세 구간**
- 대형 이벤트(10만원 경품): 22,000원 제세공과금 발생

### 7.3 개인정보 처리

- 네이버 ID 수집·처리 → 개인정보보호법 동의 절차 필요
- 제3자(다우기술) 제공 동의 별도 취득
