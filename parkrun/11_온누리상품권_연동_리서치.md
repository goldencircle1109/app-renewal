## 3. 기업구매 플랫폼 (onnuribiz.komscopay.com)

### 3.1 가입 절차

1. onnuribiz.komscopay.com 접속
2. **기업 회원 가입** — 사업자등록번호 입력
3. 서류 제출 — 사업자등록증 사본
4. 계좌 등록 — **예금주 = 사업자 상호명** 일치 필수
5. 담당자 인증 (공동인증서/휴대폰)
6. 승인 완료 → 구매 가능

### 3.2 구매 조건

| 항목 | 내용 |
|------|------|
| 구매 방법 | 계좌이체, 법인카드 (BC카드 등) |
| **기업 할인율** | **0%** (2025.10~ 할인 중단, 면가 구매) |
| 개인 할인율 | 7% (명절 10%) |
| 세금계산서 | **발행 가능** |
| 세액공제 | 구매액의 **1%** (조세특례제한법) |
| 복리후생비 손금 | 직원 1인당 연 **240만원** 한도 |

> **중요**: 2025년 10월부터 기업 할인이 중단되어 면가 구매입니다.

### 3.3 선물하기 한도

| 항목 | 한도 | 기준 |
|------|------|------|
| 발송 한도 | **월 200만원** | 1인당 |
| **수령 한도** | **월 100만원** | 1인당 |

## 4. 관광공사→라브→사용자 배포 구조

### 4.1 플로우

```
① 한국관광공사: ESG 캠페인 예산을 라이트브라더스에 입금
     ↓
② 라이트브라더스: onnuribiz.komscopay.com에서 면가 대량 구매
     ↓
③ 사용자: 앱에서 SSP → 온누리상품권 교환 요청
     ↓
④ 라이트브라더스: 선물하기로 사용자에게 발송
     ↓
⑤ 사용자: 디지털온누리 앱에서 수령 → 가맹점 결제
```

### 4.2 운영 방식 (현 시점)

- 대량 자동 API 발송: **미확인** (공개 API 없음)
- 현실적 방식: **배치 처리** (주간/월간 단위 집계 → 수동 일괄 발송)
- 향후: 조폐공사/핑거와 API 파트너십 협의 필요

## 5. API 연동 가능성

### 5.1 현재 공개 API

| API | 내용 | 트랜잭션 |
|-----|------|---------|
| 조폐공사 Open API (komsco.com/kor/contents/48) | 가맹점 정보 조회 | 읽기 전용 |
| 공공데이터포털 (data.go.kr) | 가맹점 현황 데이터 | 읽기 전용 |

→ **발급/충전/선물 관련 트랜잭션 API는 공개되어 있지 않음**

### 5.2 API 파트너십 추진 경로

1. **한국조폐공사 사업협력팀** 직접 접촉 (komsco.com)
2. **소상공인시장진흥공단** 통해 접촉 (1357)
3. **한국관광공사가 소진공/중기부에 공문 발송** (가장 효과적)
4. **핑거(FINGER)** 직접 접촉 (현재 기술 파트너)

## 6. 법적/규제 분석

### 6.1 근거 법령

- **전통시장 및 상점가 육성을 위한 특별법**
- **온누리상품권 사업 운영요령** (중기부 고시)
- **조세특례제한법** (기업 세액공제)

### 6.2 인센티브 지급 적법성

| 항목 | 판단 |
|------|------|
| 기업→직원 복리후생 지급 | **적법** (기업구매 플랫폼 설계 의도) |
| 기업→고객 인센티브 지급 | **적법** (공공기관 경품 지급 관행) |
| 제3자 대행 구매·배포 | **확인 필요** (재판매 해당 여부) |

### 6.3 리스크 체크리스트

| 리스크 | 위험도 | 대응 |
|--------|--------|------|
| '재판매' 해당 여부 | **중** | 관광공사 → 소진공 공문으로 승인 획득 |
| 수령 한도 초과 (100만/월) | **높음** | 시스템에서 사용자별 한도 관리 |
| 부정유통 과징금 (3배) | **높음** | 적법한 운영 구조 사전 확보 |
| 수신자 앱 설치 필요 | **운영 제약** | 디지털온누리 앱 설치 유도 |

### 6.4 부정유통 방지 강화 (2025년 12월 개정)

- 사용자의 재판매 행위 명시적 금지
- 부당이득 최대 **3배 과징금**
- 가맹점 매출액 기준 도입
- 부정유통 신고센터 설치 및 포상금 제도

## 7. 온누리상품권 사용처

### 7.1 사용 가능 업종 (2024.09 시행령 개정 후)

28종 제한업종을 제외한 **모든 업종** 가맹 가능:
- 신규 허용: 태권도/요가/필라테스, 의원/한의원, 악기/미술학원, 노래연습장 등
- 여전히 제한: 유흥업소, 사행성 업종 등

### 7.2 자전거 관광 연계

- 전통시장·상점가 내 자전거 판매·수리점 → 가맹점 등록 가능
- 대표코스 60선 주변 전통시장 연계 마케팅 가능
- 현재 온누리상품권-자전거 관광 연계 공식 프로그램 없음 → **라이트브라더스 신규 제안 영역**

## 8. 공공기관 배포 사례

### 8.1 서울시 에코마일리지

- 에너지 절약 목표 달성 시 온누리상품권 지급
- 2018년 사례: 1,105건, 5,525매 지류형 상품권 배포
- 현재: 서울페이/온누리상품권/관리비납부/가스캐시 중 선택

### 8.2 공무원 맞춤형 복지

- 복지포인트 중 일정 비율을 온누리상품권으로 의무 구매하는 기관 존재
- 기관 일괄구매 후 직원 배포 방식
